# 专业术语对照表

这份表用于统一笔记里的专业术语。左列保留 CAIE A Level syllabus 和教材中常见的标准英文写法，中文列使用中国大陆教材和科普语境中更常见、较规范的名称。

## 使用规则

- 讲解、说明、总结和自学笔记中，优先使用中文术语。
- 第一次出现、容易歧义或与教材英文强绑定的术语，可以写成“中文（English）”。
- 题目原文、作业、教材引用、PDF 标题、文件名、公式符号、代码、路径和命令保持英文或原样。
- 同一篇笔记里同一术语尽量保持一种写法，不在“速度 / 速率”“电势差 / 电压”等近义词之间随意切换。
- Obsidian 文件名继续使用英文；正文显示文字可以使用中文或“中文（English）”。
