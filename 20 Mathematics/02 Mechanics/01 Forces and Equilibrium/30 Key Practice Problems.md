# Forces and Equilibrium Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Forces, weight, normal contact force, tension, friction, and limiting equilibrium. State assumptions, choose a method, and check the result.

## Problem 2

## Problem 3

## Problem 4

## Problem 5
