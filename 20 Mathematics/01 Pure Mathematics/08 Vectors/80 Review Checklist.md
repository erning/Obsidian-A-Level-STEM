# Vectors Review Checklist

Use this after the worked examples and practice problems. Revisit [[20 Mathematics/01 Pure Mathematics/08 Vectors/10 Lecture Notes|Vectors Lecture Notes]] if any item is uncertain.

## Ideas to Recall

- [ ] A position vector locates a point from the origin; a displacement vector compares two points.
- [ ] A vector line has the form $\mathbf r=\mathbf a+t\mathbf b$, where $\mathbf a$ is a point on the line and $\mathbf b$ is a direction vector.
- [ ] Two 3D lines may be parallel, identical, intersecting, or skew; non-parallel lines do not automatically intersect.
- [ ] The scalar product gives angles, perpendicularity, scalar projections, and feet of perpendiculars to lines.
- [ ] A plane can be written using a normal vector, a Cartesian equation, or two independent direction vectors.
- [ ] The vector product gives a vector perpendicular to two directions and its magnitude gives parallelogram area.
- [ ] 9709 stops before shortest distance between skew lines; 9231 includes skew-line distance, common perpendicular ideas, planes, and vector product methods.

## Skills to Perform

- [ ] Find $\overrightarrow{AB}$ from two position vectors and calculate its magnitude and unit vector.
- [ ] Write a vector equation of a line from two points or from one point and one direction vector.
- [ ] Test whether a point lies on a vector line by using one parameter consistently in all components.
- [ ] Classify two vector lines by comparing direction vectors and solving component equations.
- [ ] Use $\mathbf a\cdot\mathbf b=|\mathbf a||\mathbf b|\cos\theta$ to find an angle between vectors or lines.
- [ ] Use a zero scalar product to prove perpendicularity.
- [ ] Find a foot of a perpendicular to a line by making the connecting vector perpendicular to the line direction.
- [ ] Compute a vector product and use it as a plane normal.
- [ ] Write plane equations in scalar product, Cartesian, and parametric forms.
- [ ] Substitute a line into a plane to find their intersection point.
- [ ] Find angles between lines and planes using the line direction and plane normal.
- [ ] Find distances from points to planes using the Cartesian distance formula.
- [ ] Find the shortest distance between skew lines using the scalar triple product.
- [ ] Find a line of intersection of two planes by solving both plane equations together.

## Common Errors to Avoid

- [ ] Subtracting position vectors in the wrong order for $\overrightarrow{AB}$.
- [ ] Treating a direction vector as if it were a point on the line.
- [ ] Checking only two component equations when testing a 3D line intersection.
- [ ] Calling non-intersecting 3D lines parallel without checking whether their directions are scalar multiples.
- [ ] Forgetting absolute values in distance and angle formulae.
- [ ] Using the angle between a line and a plane's normal as the line-plane angle.
- [ ] Using vector product in 9709 work where the syllabus only requires scalar product methods.
- [ ] Assuming shortest distance between skew lines is part of 9709 Pure Mathematics.

## Ready to Move On When

- [ ] I can state what each vector in a calculation represents before doing algebra.
- [ ] I can choose between scalar product, vector product, substitution, and simultaneous equations for a 3D geometry problem.
- [ ] I can explain why a pair of lines is parallel, identical, intersecting, or skew.
- [ ] I can convert between vector and Cartesian descriptions of a plane.
- [ ] I can check every final point by substituting it back into the original line or plane equations.
