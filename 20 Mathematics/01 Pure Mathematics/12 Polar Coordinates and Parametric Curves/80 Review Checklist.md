# Polar Coordinates and Parametric Curves Review Checklist

Use this checklist after the practice problems. Any unchecked item is a cue to return to the lecture notes or worked examples.

## Ideas to Recall

- [ ] Relations between Cartesian and polar coordinates.
- [ ] Sketching simple polar curves and identifying symmetry, intersections, and least or greatest radius.
- [ ] Area in polar coordinates using $\frac{1}{2}\int r^2\,d\theta$.
- [ ] Parametric curves, derivatives, tangents, and graph features.
- [ ] Choosing the representation that makes a curve easier to understand.

## Skills to Perform

- [ ] Convert between Cartesian and polar equations.
- [ ] Sketch polar and parametric curves.
- [ ] Find tangents and enclosed areas.

## Common Errors to Avoid

- [ ] Using the wrong angle interval.
- [ ] Forgetting that one point can have multiple polar descriptions.
- [ ] Treating parameter $t$ as if it must be time.
