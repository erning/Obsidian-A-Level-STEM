# Polar Coordinates and Parametric Curves Review Checklist

Use this after the worked examples and practice problems. Revisit [[20 Mathematics/01 Pure Mathematics/12 Polar Coordinates and Parametric Curves/10 Lecture Notes|Polar Coordinates and Parametric Curves Lecture Notes]] if any item is uncertain.

## Ideas to Recall

- [ ] Polar coordinates use $x=r\cos\theta$, $y=r\sin\theta$, and $r^2=x^2+y^2$.
- [ ] In the 9231 polar-coordinate convention used here, take $r\ge0$ unless a question says otherwise.
- [ ] A point can have more than one polar description because angles can differ by $2\pi$.
- [ ] A polar sketch should show symmetry, intercepts, pole behaviour, and least or greatest values of $r$.
- [ ] Polar area is $A=\frac12\int_\alpha^\beta r^2\,d\theta$.
- [ ] Parametric equations keep a parameter interval; eliminating the parameter does not remove that restriction.
- [ ] For parametric curves, $\frac{dy}{dx}=\frac{dy/dt}{dx/dt}$ when $\frac{dx}{dt}\ne0$.
- [ ] Horizontal tangents need $\frac{dy}{dt}=0$ and $\frac{dx}{dt}\ne0$; vertical tangents need $\frac{dx}{dt}=0$ and $\frac{dy}{dt}\ne0$.

## Skills to Perform

- [ ] Convert points between Cartesian and polar coordinates, checking the quadrant.
- [ ] Convert equations using $x=r\cos\theta$, $y=r\sin\theta$, and $r^2=x^2+y^2$.
- [ ] Recognise simple polar equations for circles and straight lines.
- [ ] Identify key values of $r$ before sketching a polar curve.
- [ ] Choose polar area limits from the sketch, not from habit.
- [ ] Solve intersections of polar curves while respecting the stated angle interval.
- [ ] Eliminate a parameter and preserve restrictions on $x$, $y$, or $t$.
- [ ] Find tangent and normal equations for parametric curves.
- [ ] Find stationary points and vertical tangents from $\frac{dx}{dt}$ and $\frac{dy}{dt}$.
- [ ] Use $\frac{d^2y}{dx^2}=\frac{d}{dt}\left(\frac{dy}{dx}\right)\div\frac{dx}{dt}$ when needed.

## Common Errors to Avoid

- [ ] Using $\tan^{-1}(y/x)$ without checking the quadrant.
- [ ] Forgetting that $\theta$ may be undefined at the pole.
- [ ] Dividing by $r$ without checking whether the pole is included.
- [ ] Using the wrong angle interval for a polar sketch or intersection.
- [ ] Double-counting polar area when a curve is traced more than once.
- [ ] Eliminating a parameter and forgetting the parameter range.
- [ ] Treating $\frac{dy}{dt}$ as the gradient in the $xy$-plane.
- [ ] Calling a tangent horizontal when both $\frac{dx}{dt}$ and $\frac{dy}{dt}$ are zero.

## Ready to Move On When

- [ ] I can choose whether Cartesian, polar, or parametric form is the clearest representation.
- [ ] I can sketch a simple polar curve from key values without excessive plotting.
- [ ] I can set polar area limits from a diagram and explain the region.
- [ ] I can find parametric tangents and normals without losing the parameter value.
- [ ] I can state the restrictions that survive after converting forms.
