## 3. Method Selection

Use these habits while studying:

- Plot sample points before trusting an unfamiliar curve.
- Track parameter or angle intervals carefully.
- Convert forms only when conversion helps the problem.

Use these practice routes to turn the ideas into fluency:

- Convert between Cartesian and polar equations.
- Sketch polar and parametric curves.
- Find tangents and enclosed areas.

When two methods seem possible, choose the one that exposes the most structure with the least algebra. Then check the answer in another representation whenever possible.

## 6. Common Traps

- Using the wrong angle interval.
- Forgetting that one point can have multiple polar descriptions.
- Treating parameter $t$ as if it must be time.
