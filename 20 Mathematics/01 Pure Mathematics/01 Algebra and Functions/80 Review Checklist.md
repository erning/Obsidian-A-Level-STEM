# Algebra and Functions Review Checklist

Use this after working through [[20 Mathematics/01 Pure Mathematics/01 Algebra and Functions/10 Lecture Notes|Lecture Notes]], [[20 Mathematics/01 Pure Mathematics/01 Algebra and Functions/20 Worked Examples|Worked Examples]], and [[20 Mathematics/01 Pure Mathematics/01 Algebra and Functions/30 Key Practice Problems|Key Practice Problems]].

## Core Skills

- [ ] I can move between standard, factorised, and completed-square forms of a quadratic.
- [ ] I can use the discriminant to decide the number of real roots.
- [ ] I can use the discriminant as a tangency or parameter condition.
- [ ] I can solve quadratic inequalities using a sign diagram.
- [ ] I can recognise equations that are quadratic in another expression.
- [ ] I can state the domain and range of simple functions.
- [ ] I can find and use composite functions.
- [ ] I can decide whether a function is one-one on a given domain.
- [ ] I can find an inverse function and state its domain.
- [ ] I can describe graph transformations using translation, reflection, and stretch.
- [ ] I can solve simple modulus equations and inequalities.
- [ ] I can use polynomial division and identify quotient and remainder.
- [ ] I can apply the factor theorem and remainder theorem.
- [ ] I can use root-coefficient relations for cubic and quartic equations.
- [ ] I can form a transformed polynomial equation from related roots.
- [ ] I can sketch rational functions using asymptotes and intercepts.
- [ ] I can find the range of a rational function using a discriminant argument.
- [ ] I can decompose rational functions into partial fractions.

## Representation Checks

- [ ] For every equation I solve, I can also describe the graph interpretation.
- [ ] For every function problem, I check the domain before manipulating symbols.
- [ ] For every inverse function, I check whether the original function is one-one.
- [ ] For every rational function, I identify excluded values before solving.
- [ ] For every partial fraction decomposition, I recombine to check.

## Ready to Move On?

You are ready to connect this topic to [[20 Mathematics/01 Pure Mathematics/06 Integration/00 Overview|Integration]] when you can:

- use partial fractions reliably;
- recognise useful algebraic substitutions;
- control domain restrictions;
- read graph features from algebraic form.

You are ready to connect it to [[20 Mathematics/01 Pure Mathematics/09 Complex Numbers/00 Overview|Complex Numbers]] when you can:

- use polynomial roots as algebraic objects;
- work with root-coefficient relations;
- accept that some polynomial equations have no real roots but still have
  algebraic structure.
