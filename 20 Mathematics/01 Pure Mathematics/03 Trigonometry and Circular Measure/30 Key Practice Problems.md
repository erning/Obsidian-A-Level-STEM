# Trigonometry and Circular Measure Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Radian measure, arc length, and sector area. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Trigonometric ratios, graphs, inverse trigonometric functions, and equations. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Compound-angle, double-angle, and related identities. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: General solutions and expressions such as $a\sin\theta + b\cos\theta$. State assumptions, choose a method, and check the result.

## Problem 5

Complete this practice task: Solve trigonometric equations over specified intervals.

## Problem 6

Complete this practice task: Use identities to simplify expressions.

## Problem 7

Complete this practice task: Apply arc length and sector area formulae.

## Problem 8

Solve a representative Trigonometry and Circular Measure problem and check it using a second representation.

## Extension
