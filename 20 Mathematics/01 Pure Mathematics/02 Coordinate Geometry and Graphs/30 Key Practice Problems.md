# Coordinate Geometry and Graphs Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Equations of straight lines in common forms. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Distances, gradients, midpoints, intersections, and relationships between parallel and perpendicular lines. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Circle equations in centre-radius and expanded forms. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Intersections of lines, circles, and curves as solutions of equations. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Graph transformations of $y = f(x)$. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Find equations of lines and circles.

## Problem 7

Complete this practice task: Solve line-circle and graph-intersection problems.

## Problem 8

Complete this practice task: Sketch transformed graphs with domain and range.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
