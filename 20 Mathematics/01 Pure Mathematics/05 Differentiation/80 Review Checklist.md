# Differentiation Review Checklist

Use this checklist after the practice problems. Any unchecked item is a cue to
return to the lecture notes or worked examples.

## Ideas to Recall

- [ ] A derivative is both a tangent gradient and an instantaneous rate of
      change.
- [ ] The second derivative describes how the gradient changes.
- [ ] Chain rule is needed for composite functions.
- [ ] Product rule is needed when two variable expressions are multiplied.
- [ ] Quotient rule uses $u'v-uv'$ in the numerator.
- [ ] Trigonometric derivatives assume angles are measured in radians.
- [ ] A tangent line needs a point and a tangent gradient.
- [ ] A normal gradient is the negative reciprocal of the tangent gradient,
      except in horizontal or vertical special cases.
- [ ] Stationary points occur where $f'(x)=0$.
- [ ] The second-derivative test is inconclusive when $f''(a)=0$.
- [ ] Implicit differentiation treats $y$ as a function of $x$.
- [ ] Parametric differentiation uses
      $\frac{dy}{dx}=\frac{dy/dt}{dx/dt}$.
- [ ] For parametric curves,
      $\frac{d^2y}{dx^2}=\frac{d}{dt}\left(\frac{dy}{dx}\right)/(dx/dt)$.
- [ ] A Maclaurin series uses derivatives evaluated at $0$.

## Skills to Perform

- [ ] Differentiate powers, roots, and reciprocal powers.
- [ ] Apply chain, product, and quotient rules cleanly.
- [ ] Differentiate $e^x$, $\ln x$, $\sin x$, $\cos x$, $\tan x$, and
      composites.
- [ ] Differentiate $\tan^{-1}x$ and the 9231 inverse trigonometric and
      hyperbolic functions when required.
- [ ] Find tangent and normal equations.
- [ ] Locate and classify stationary points.
- [ ] Determine increasing and decreasing intervals from the sign of $f'(x)$.
- [ ] Solve connected rates of change using the chain rule.
- [ ] Form and optimise a one-variable function from a geometric context.
- [ ] Differentiate implicit equations and use the gradient in line problems.
- [ ] Differentiate parametric curves and compute second derivatives.
- [ ] Build the first few terms of a Maclaurin series from successive
      derivatives.

## Common Errors to Avoid

- [ ] Forgetting the derivative of the inner function in the chain rule.
- [ ] Reversing the numerator in the quotient rule.
- [ ] Differentiating $\ln x$ without noting $x>0$.
- [ ] Treating $f'(x)=0$ as automatically meaning a maximum or minimum.
- [ ] Using the second-derivative test when it is inconclusive.
- [ ] Finding a gradient but forgetting the point needed for a tangent or
      normal.
- [ ] Using $\frac{dy}{dt}$ as though it were $\frac{dy}{dx}$.
- [ ] Forgetting $\frac{dy}{dx}$ factors in implicit differentiation.
- [ ] Computing parametric $\frac{d^2y}{dx^2}$ as
      $\frac{d^2y/dt^2}{d^2x/dt^2}$.
- [ ] Losing units or signs in connected rates of change.
- [ ] Ignoring domain restrictions for inverse or hyperbolic derivatives.

## Ready to Move On When

- [ ] I can explain what a derivative means on a graph and in a rate problem.
- [ ] I can choose the correct differentiation rule from the function
      structure.
- [ ] I can find and use first and second derivatives.
- [ ] I can write tangents and normals without losing the point of contact.
- [ ] I can classify stationary points and handle inconclusive second
      derivatives.
- [ ] I can solve optimisation and connected-rate problems in context.
- [ ] I can differentiate implicit and parametric relations.
- [ ] I can use the 9231 extension derivatives and Maclaurin formula when
      needed.
- [ ] I can connect this topic to [Integration](../06%20Integration/00%20Overview.md),
      [Further Calculus and Differential Equations](../13%20Further%20Calculus%20and%20Differential%20Equations/00%20Overview.md),
      and [Kinematics and Newtonian Motion](../../02%20Mechanics/02%20Kinematics%20and%20Newtonian%20Motion/00%20Overview.md).
