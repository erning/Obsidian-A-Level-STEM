# Further Calculus and Differential Equations Review Checklist

Use this after the worked examples and practice problems. Revisit [Further Calculus and Differential Equations Lecture Notes](10%20Lecture%20Notes.md) if any item is uncertain.

## Ideas to Recall

- [ ] Hyperbolic functions are defined using exponentials.
- [ ] $\cosh^2x-\sinh^2x=1$ is the central hyperbolic identity.
- [ ] Maclaurin series come from derivatives at $x=0$.
- [ ] Further integration often begins by recognising a structure: square root, product, recurrence, arc length, or substitution.
- [ ] Separable differential equations put all $y$ terms with $dy$ and all $x$ terms with $dx$.
- [ ] A first-order linear equation must be in the form $\frac{dy}{dx}+P(x)y=Q(x)$ before using an integrating factor.
- [ ] A constant-coefficient linear differential equation uses a complementary function plus a particular integral.
- [ ] Initial conditions select constants from the general solution.
- [ ] A model solution should be interpreted, not only solved symbolically.

## Skills to Perform

- [ ] Use exponential definitions to prove hyperbolic identities.
- [ ] Solve simple equations involving $\sinh x$ and $\cosh x$ using $u=e^x$.
- [ ] Differentiate hyperbolic and inverse hyperbolic functions.
- [ ] Find the first few Maclaurin terms of a function.
- [ ] Recognise and integrate forms such as $\frac{1}{\sqrt{a^2-x^2}}$.
- [ ] Use a reduction formula to evaluate a definite integral.
- [ ] Find arc length for Cartesian or parametric curves when required.
- [ ] Solve separable differential equations and apply initial conditions.
- [ ] Use an integrating factor for first-order linear differential equations.
- [ ] Solve second-order constant-coefficient equations with distinct, repeated, or complex auxiliary roots.
- [ ] Choose a suitable particular integral for a polynomial, exponential, sine, or cosine forcing term.
- [ ] Use a given substitution to reduce a differential equation to a standard type.

## Common Errors to Avoid

- [ ] Dropping the arbitrary constant after integration.
- [ ] Separating variables before checking that the equation is actually separable.
- [ ] Forgetting special constant solutions when division by a function of $y$ is used.
- [ ] Using an integrating factor before making the coefficient of $\frac{dy}{dx}$ equal to $1$.
- [ ] Confusing the complementary function with the particular integral.
- [ ] Choosing a particular-integral trial form that duplicates part of the complementary function.
- [ ] Applying an initial condition before obtaining the general solution.
- [ ] Giving a model answer without interpreting the limiting value or constant.

## Ready to Move On When

- [ ] I can classify a differential equation before solving it.
- [ ] I can solve separable, first-order linear, and constant-coefficient equations.
- [ ] I can use hyperbolic definitions and identities without treating them as trigonometric copies.
- [ ] I can choose an integration method from the visible structure of the integrand.
- [ ] I can check a DE solution by differentiating and substituting back.
