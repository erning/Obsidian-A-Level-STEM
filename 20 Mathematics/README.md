# CAIE Mathematics 9709 and Further Mathematics 9231

这个目录把 CAIE Mathematics 9709 和 CAIE Further Mathematics 9231 合并为一条数学学习路线。9709 作为基础数学主线，9231 作为进阶延伸；学习时按概念依赖来组织，而不是把它们拆成两个互相隔离的科目。

这样更适合兴趣学习：代数、函数、微积分、向量、力学、概率与统计本来就是连续发展的数学语言。9231 的很多内容不是另一门孤立的数学，而是在 9709 的基础上继续提高抽象程度、计算能力和建模能力。

## 学习原则

- 先用 9709 建立可靠的基础：函数、三角、微积分、向量、概率统计和力学。
- 再用 9231 加深结构理解：复数、矩阵、级数、进一步微积分、微分方程、更深入的力学与统计。
- 笔记按主题组织，避免在 9709 和 9231 之间重复建立同一份概念笔记。
- 遇到物理里用到的数学工具，优先和 [[10 Physics/04 Reference/Math Tools and Formulae|Physics 数学工具与公式表]] 互相链接。
