# Sampling, Estimation and Hypothesis Tests Review Checklist

Use this after the worked examples and practice problems. Revisit [[20 Mathematics/03 Probability and Statistics/06 Sampling Estimation and Hypothesis Tests/10 Lecture Notes|Sampling, Estimation and Hypothesis Tests Lecture Notes]] if any item is uncertain.

## Ideas to Recall

- [ ] A population parameter describes the population; a statistic is calculated from the sample.
- [ ] Randomness matters because it gives a statistic a sampling distribution.
- [ ] $E(\bar X)=\mu$ and $\operatorname{Var}(\bar X)=\frac{\sigma^2}{n}$.
- [ ] The standard error of $\bar X$ is $\frac{\sigma}{\sqrt n}$ when $\sigma$ is known.
- [ ] The Central Limit Theorem justifies approximate normality for many large-sample means.
- [ ] An unbiased variance estimate divides by $n-1$.
- [ ] A confidence interval describes the long-run method, not the probability that one fixed interval contains the parameter.
- [ ] A significance level is the probability of rejecting $H_0$ when $H_0$ is true.
- [ ] A p-value is calculated assuming $H_0$ is true.
- [ ] Type I error means rejecting a true $H_0$; Type II error means not rejecting a false $H_0$.
- [ ] 9231 extends 9709 inference to $t$-tests, paired tests, two-sample methods, pooled variance, and $t$ confidence intervals.

## Skills to Perform

- [ ] Identify the population, parameter, sample, statistic, estimator, and estimate.
- [ ] Explain why a sampling method may be biased or unrepresentative.
- [ ] Find the sampling distribution, variance, and standard error of $\bar X$.
- [ ] Calculate unbiased estimates of mean and variance from raw data.
- [ ] Construct and interpret confidence intervals for a mean.
- [ ] Construct and interpret large-sample confidence intervals for a proportion.
- [ ] Write $H_0$ and $H_1$ before calculating.
- [ ] Choose the correct tail direction for a hypothesis test.
- [ ] Use direct binomial or Poisson probabilities for small discrete tests.
- [ ] Use normal approximations with continuity correction where appropriate.
- [ ] Carry out normal tests for population means with known variance or large samples.
- [ ] Calculate Type I and Type II error probabilities from a stated critical region.
- [ ] Use a $t$-test or $t$ interval when the sample is small, normality is assumed, and variance is unknown.
- [ ] Calculate a pooled variance estimate when a common variance is assumed.

## Common Errors to Avoid

- [ ] Treating a convenience sample as if it were random.
- [ ] Confusing population standard deviation, sample standard deviation, and standard error.
- [ ] Using $n$ instead of $n-1$ for the unbiased variance estimate.
- [ ] Saying the null hypothesis is proved or accepted.
- [ ] Interpreting the significance level as the probability that $H_0$ is true.
- [ ] Reversing one-tailed and two-tailed alternatives.
- [ ] Using a two-tailed critical value for a one-tailed test.
- [ ] Forgetting continuity correction in a normal approximation to a binomial or Poisson test.
- [ ] Calculating a Type II error without specifying an alternative parameter value.
- [ ] Using a normal test for a small sample when a $t$-test is required.

## Ready to Move On When

- [ ] I can describe the population and parameter before writing formulae.
- [ ] I can distinguish estimation questions from testing questions.
- [ ] I can build a confidence interval and state what it means in context.
- [ ] I can carry out a hypothesis test using a p-value or a critical region consistently.
- [ ] I can choose between binomial, Poisson, normal, and $t$ methods from the assumptions.
- [ ] I can compute and interpret Type I and Type II error probabilities.
- [ ] I can state a conclusion without claiming that $H_0$ has been proved.
