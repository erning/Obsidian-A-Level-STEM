# Counting and Probability Review Checklist

Use this checklist after the practice problems. Any unchecked item is a cue to return to the lecture notes or worked examples.

## Ideas to Recall

- [ ] Permutations and combinations.
- [ ] Sample space, events, complements, mutually exclusive events, and independent events.
- [ ] Addition and multiplication rules.
- [ ] Conditional probability.
- [ ] Tree diagrams, Venn diagrams, and probability notation.

## Skills to Perform

- [ ] Count arrangements and selections.
- [ ] Use conditional probability.
- [ ] Solve multi-stage probability problems.

## Common Errors to Avoid

- [ ] Using permutations when combinations are needed.
- [ ] Assuming events are independent without evidence.
- [ ] Counting the same outcome twice.

## Ready to Move On When

- [ ] I can explain the purpose of Counting and Probability without reading the notes.
- [ ] I can choose a method before seeing a worked solution.
- [ ] I can check my answer using units, signs, domains, graph features, limiting cases, or context.
- [ ] I can connect this topic to at least one neighbouring topic.
