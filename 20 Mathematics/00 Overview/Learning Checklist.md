# 学习内容清单

这份清单回答两个问题：**要学哪些东西**，以及每类内容**应该学到什么程度**。勾选不代表“做完题”，而是代表你能解释概念、选择方法、画图检查，并把结果放回问题语境。

## 纯数学

### [[20 Mathematics/01 Pure Mathematics/01 Algebra and Functions|代数与函数]]

- [ ] 会处理多项式、方程、不等式、模函数和代数分式。
- [ ] 理解函数、定义域、值域、复合函数和反函数。
- [ ] 能用图像理解方程的根、交点和参数影响。
- [ ] 会用配方法、判别式和因式定理。
- [ ] 能把代数技巧用于建模，而不是只做符号整理。

### [[20 Mathematics/01 Pure Mathematics/02 Coordinate Geometry and Graphs|坐标几何与图像]]

- [ ] 会求直线方程、距离、中点、交点和平行/垂直关系。
- [ ] 能从图像读斜率、截距、渐近线、对称性和交点。
- [ ] 会处理曲线与直线相交的问题。
- [ ] 能把代数图像变换和函数表达式对应起来。

### [[20 Mathematics/01 Pure Mathematics/03 Trigonometry and Circular Measure|三角函数与弧度制]]

- [ ] 会用度和弧度，并能解释为什么微积分中优先用弧度。
- [ ] 会使用弧长、扇形面积和基本三角图像。
- [ ] 掌握三角恒等式、和差角、倍角和通解。
- [ ] 能把三角函数看成周期模型，而不是只看成三角形计算。

### [[20 Mathematics/01 Pure Mathematics/04 Sequences Series and Binomial Expansions|数列、级数与二项展开]]

- [ ] 会处理等差数列、等比数列和有限/无限级数。
- [ ] 会用求和符号和求和公式。
- [ ] 理解二项式展开及其适用条件。
- [ ] 能把级数和近似、证明、概率连接起来。

### [[20 Mathematics/01 Pure Mathematics/05 Differentiation|微分法]]

- [ ] 理解导数是瞬时变化率和切线斜率。
- [ ] 会求多项式、指数、对数、三角、隐函数导数和参数方程导数。
- [ ] 会用乘积法则、商法则和链式法则。
- [ ] 能用导数分析递增/递减、驻点、凹凸性和优化问题。
- [ ] 能把导数连接到速度、加速度、边际变化和曲线草图。

### [[20 Mathematics/01 Pure Mathematics/06 Integration|积分法]]

- [ ] 理解积分是反导数和累积量。
- [ ] 会求面积、体积、概率密度下的概率和物理中的位移/功。
- [ ] 会用换元法、分部积分法、部分分式和三角恒等式。
- [ ] 理解反常积分、数值变量积分和模型误差。

### [[20 Mathematics/01 Pure Mathematics/07 Logarithms Exponentials and Numerical Methods|对数、指数与数值方法]]

- [ ] 理解对数和指数的反函数关系。
- [ ] 会用 $e^x$、$\ln x$ 和对数法则解方程。
- [ ] 能用指数函数建模增长、衰减和比例变化。
- [ ] 会用符号变化、不动点迭代和图像定位近似根。

### [[20 Mathematics/01 Pure Mathematics/08 Vectors|向量]]

- [ ] 会做二维和三维向量运算。
- [ ] 理解点积、叉积、投影和角度。
- [ ] 会写直线和平面的矢量方程。
- [ ] 会求点到线/平面的距离，以及线面、面面关系。
- [ ] 能把向量用于力学和物理的空间建模。

### [[20 Mathematics/01 Pure Mathematics/09 Complex Numbers|复数]]

- [ ] 会在笛卡尔形式、绝对值-辐角形式和指数形式之间转换。
- [ ] 理解复平面的几何意义。
- [ ] 会使用棣莫弗定理、单位根和复数根。
- [ ] 能把复数与三角学、旋转和微分方程联系起来。

### [[20 Mathematics/01 Pure Mathematics/10 Matrices and Transformations|矩阵与变换]]

- [ ] 会做矩阵加法、乘法、逆矩阵、行列式和联立方程。
- [ ] 理解矩阵作为线性变换。
- [ ] 会处理特征值、特征向量和对角化。
- [ ] 理解 Cayley-Hamilton 定理的作用。

### [[20 Mathematics/01 Pure Mathematics/11 Proof and Mathematical Induction|证明与数学归纳法]]

- [ ] 理解证明不是计算附属品，而是确认结构的方式。
- [ ] 会写归纳法的基础情形、归纳假设和归纳步骤。
- [ ] 能用归纳法证明求和公式、整除性和递推关系。
- [ ] 会区分例子、模式和证明。

### [[20 Mathematics/01 Pure Mathematics/12 Polar Coordinates and Parametric Curves|极坐标与参数曲线]]

- [ ] 会在笛卡尔、极坐标和参数形式之间转换。
- [ ] 能画极坐标曲线草图，并理解对称性和范围。
- [ ] 会求极坐标面积、参数曲线斜率、弧长和曲面面积。
- [ ] 能把参数看成描述运动或生成曲线的变量。

### [[20 Mathematics/01 Pure Mathematics/13 Further Calculus and Differential Equations|进阶微积分与微分方程]]

- [ ] 会处理反三角函数、双曲函数和反双曲函数。
- [ ] 会使用递推公式、麦克劳林级数和进阶积分技巧。
- [ ] 会建立并求解可分离变量、一阶和二阶线性微分方程。
- [ ] 能把微分方程连接到增长、衰减、振动和力学。

## 力学

### [[20 Mathematics/02 Mechanics/01 Forces and Equilibrium|力与平衡]]

- [ ] 会画受力图。
- [ ] 会分解力，处理平衡和斜面问题。
- [ ] 理解摩擦力、支持力、张力和重力。
- [ ] 能用力矩分析刚体平衡。

### [[20 Mathematics/02 Mechanics/02 Kinematics and Newtonian Motion|运动学与牛顿运动]]

- [ ] 会用位移、速度、加速度和图像描述一维运动。
- [ ] 会用恒定加速度方程。
- [ ] 理解牛顿运动定律，并能处理连接体。
- [ ] 会把微积分形式的运动关系用于一般运动。

### [[20 Mathematics/02 Mechanics/03 Momentum Collisions and Impulse|动量、碰撞与冲量]]

- [ ] 理解动量、冲量和动量守恒。
- [ ] 会处理正碰、斜碰和恢复系数。
- [ ] 能判断系统边界和外力影响。

### [[20 Mathematics/02 Mechanics/04 Work Energy Power and Elasticity|功、能量、功率与弹性]]

- [ ] 理解功、动能、引力势能和功率。
- [ ] 会用能量守恒和功-能量原理。
- [ ] 理解胡克定律、弹性模量和弹性势能。
- [ ] 能用能量方法简化复杂运动。

### [[20 Mathematics/02 Mechanics/05 Projectiles and Circular Motion|抛体与圆周运动]]

- [ ] 会把抛体运动分解为水平和竖直方向。
- [ ] 理解水平/竖直圆周运动的加速度和受力。
- [ ] 会判断张力、法向力和完整圆周运动条件。
- [ ] 能把圆周运动和物理的场、简谐运动（SHM）连接起来。

### [[20 Mathematics/02 Mechanics/06 Rigid Bodies and Variable Forces|刚体与变力]]

- [ ] 会用质心和力矩处理刚体。
- [ ] 会建立变量力下的微分方程。
- [ ] 能根据 $a = dv/dt$ 或 $a = v\,dv/dx$ 选择方程形式。
- [ ] 理解模型限制，例如粒子模型、刚体模型和无空气阻力假设。

## 概率与统计

### [[20 Mathematics/03 Probability and Statistics/01 Data Representation and Summary|数据表示与概括]]

- [ ] 会选择合适图表表示数据。
- [ ] 会计算和解释均值、中位数、众数、四分位数、标准差。
- [ ] 会读直方图、箱线图、茎叶图和累积频数图。
- [ ] 能判断图表是否误导。

### [[20 Mathematics/03 Probability and Statistics/02 Counting and Probability|计数与概率]]

- [ ] 会用阶乘、排列和组合。
- [ ] 理解独立、互斥、条件概率和树状图。
- [ ] 会把随机过程拆成清晰事件。
- [ ] 能检查概率和是否为 1。

### [[20 Mathematics/03 Probability and Statistics/03 Discrete Random Variables|离散随机变量]]

- [ ] 理解离散随机变量、概率分布、期望和方差。
- [ ] 会使用二项分布、几何分布和简单离散模型。
- [ ] 会处理随机变量的函数。
- [ ] 能解释参数对分布形状的影响。

### [[20 Mathematics/03 Probability and Statistics/04 Normal and Poisson Distributions|正态分布与泊松分布]]

- [ ] 理解正态分布的标准化和面积意义。
- [ ] 会用正态近似检查离散模型。
- [ ] 理解泊松分布的事件率模型。
- [ ] 会处理泊松分布加法、近似和简单检验。

### [[20 Mathematics/03 Probability and Statistics/05 Continuous Random Variables|连续随机变量]]

- [ ] 理解概率密度函数和分布函数。
- [ ] 会求概率、期望、方差、中位数和众数。
- [ ] 会处理变换 $Y = g(X)$ 的密度。
- [ ] 能用积分解释连续概率。

### [[20 Mathematics/03 Probability and Statistics/06 Sampling Estimation and Hypothesis Tests|抽样、估计与假设检验]]

- [ ] 理解抽样、估计量、置信区间和标准误。
- [ ] 理解原假设、备择假设、显著性水平和拒绝域。
- [ ] 会解释第一类错误和第二类错误。
- [ ] 能用正态、t 和二项分布语境进行推断。

### [[20 Mathematics/03 Probability and Statistics/07 Chi Squared Nonparametric and PGF|卡方、非参数检验与 PGF]]

- [ ] 会用卡方检验做拟合优度和独立性。
- [ ] 理解符号检验、Wilcoxon 符号秩检验和秩和检验的使用场景。
- [ ] 理解概率母函数。
- [ ] 会用 PGF 求和、均值和方差。
