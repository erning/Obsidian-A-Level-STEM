# 学习内容清单

这份清单回答两个问题：**要学哪些东西**，以及每类内容**应该学到什么程度**。勾选不代表“做完题”，而是代表你能解释、计算、画图、联系实验，并知道模型边界。

## 基础工具

### [[10 Physics/01 Topics/01 Physical Quantities and Units|Physical Quantities and Units]]

- [ ] 理解物理量由数值和单位组成。
- [ ] 会用 SI base units 和 derived units。
- [ ] 会用量纲检查公式是否可能成立。
- [ ] 会处理数量级估算和常用 prefix。
- [ ] 理解 systematic error、random error、precision、accuracy。
- [ ] 会估算和传播简单不确定度。
- [ ] 会区分 scalar 和 vector，并做矢量合成、相减和分解。

### [[10 Physics/04 Reference/Math Tools and Formulae|数学工具与公式表]]

- [ ] 会代数变形，能把目标量变成公式主语。
- [ ] 会用三角函数和矢量分解处理方向问题。
- [ ] 会从图像读 gradient、intercept、area 和 tangent。
- [ ] 会处理指数、对数和线性化。
- [ ] 每条公式都能说出适用条件和单位。

## 力学与材料

### [[10 Physics/01 Topics/02 Kinematics|Kinematics]]

- [ ] 用 distance、displacement、speed、velocity、acceleration 描述运动。
- [ ] 在文字、公式和图像之间转换。
- [ ] 用匀加速模型处理直线运动和自由落体。
- [ ] 把 projectile motion 分解为水平和竖直方向。
- [ ] 知道空气阻力什么时候会破坏模型。

### [[10 Physics/01 Topics/03 Dynamics|Dynamics]]

- [ ] 用 Newton's laws 解释运动变化。
- [ ] 会画 free-body diagram。
- [ ] 用 resultant force 连接 acceleration。
- [ ] 用 momentum 和 impulse 理解碰撞。
- [ ] 判断一个系统是否可用 momentum conservation。
- [ ] 用 terminal velocity 解释阻力与重力的平衡。

### [[10 Physics/01 Topics/04 Forces Density and Pressure|Forces, Density and Pressure]]

- [ ] 用 moment 和 torque 理解转动趋势。
- [ ] 理解平衡需要合力和合力矩同时为零。
- [ ] 用 density 和 pressure 描述连续物质。
- [ ] 从 hydrostatic pressure 理解深度、密度和压强差。
- [ ] 用 Archimedes' principle 理解浮力。

### [[10 Physics/01 Topics/05 Work Energy and Power|Work, Energy and Power]]

- [ ] 用 work 连接 force 和 energy transfer。
- [ ] 用 conservation of energy 简化力学问题。
- [ ] 理解 kinetic energy 和 gravitational potential energy。
- [ ] 用 power 描述能量转移速率。
- [ ] 会解释效率和能量耗散。

### [[10 Physics/01 Topics/06 Deformation of Solids|Deformation of Solids]]

- [ ] 理解 Hooke's law 和 spring constant。
- [ ] 用 stress、strain、Young modulus 描述材料。
- [ ] 从 force-extension 和 stress-strain 图像读材料性质。
- [ ] 区分 elastic deformation 和 plastic deformation。
- [ ] 用图像面积理解 elastic potential energy。

### [[10 Physics/01 Topics/12 Motion in a Circle|Motion in a Circle]]

- [ ] 用 radian、angular speed 和 period 描述圆周运动。
- [ ] 理解 centripetal acceleration 的方向和来源。
- [ ] 知道 centripetal force 是合力角色，不是一种新力。
- [ ] 能把 circular motion 连接到 gravity、magnetism 和 oscillations。

### [[10 Physics/01 Topics/13 Gravitational Fields|Gravitational Fields]]

- [ ] 把 gravity 理解为 field。
- [ ] 使用 Newton's law of gravitation。
- [ ] 用 circular motion 分析轨道。
- [ ] 理解 gravitational field strength 和 gravitational potential。
- [ ] 会比较 gravitational field 与 electric field。

## 波、叠加与振动

### [[10 Physics/01 Topics/07 Waves|Waves]]

- [ ] 用 amplitude、period、frequency、wavelength、phase 描述波。
- [ ] 理解 progressive wave 传递能量。
- [ ] 区分 transverse 和 longitudinal waves。
- [ ] 用 Doppler effect 解释相对运动导致的频率变化。
- [ ] 理解 electromagnetic spectrum 和 polarisation。

### [[10 Physics/01 Topics/08 Superposition|Superposition]]

- [ ] 理解 superposition 是波现象的核心规则。
- [ ] 用 path difference 和 phase 理解 interference。
- [ ] 理解 diffraction 与 aperture size 的关系。
- [ ] 用 nodes 和 antinodes 理解 stationary waves。
- [ ] 用 diffraction grating 理解光谱测量。

### [[10 Physics/01 Topics/17 Oscillations|Oscillations]]

- [ ] 理解 SHM 的 restoring acceleration。
- [ ] 用图像理解 displacement、velocity、acceleration 的相位关系。
- [ ] 理解 SHM 中能量的来回转化。
- [ ] 区分 damping、forced oscillation 和 resonance。
- [ ] 能把 oscillations 与 waves、circuits 和 real-world resonance 联系起来。

## 电、磁与场

### [[10 Physics/01 Topics/09 Electricity|Electricity]]

- [ ] 从 charge carriers 的运动理解 current。
- [ ] 用 energy per unit charge 理解 potential difference。
- [ ] 用 power 解释电路中的能量转移。
- [ ] 理解 resistance、resistivity 和 I-V characteristics。
- [ ] 解释 LDR、thermistor、diode 和 filament lamp 的行为。

### [[10 Physics/01 Topics/10 DC Circuits|DC Circuits]]

- [ ] 用 charge conservation 理解 Kirchhoff's first law。
- [ ] 用 energy conservation 理解 Kirchhoff's second law。
- [ ] 会分析 series、parallel 和 mixed circuits。
- [ ] 理解 e.m.f.、internal resistance 和 terminal p.d.。
- [ ] 用 potential divider 理解传感器电路。

### [[10 Physics/01 Topics/18 Electric Fields|Electric Fields]]

- [ ] 把 electric field 理解为每一点的 force rule。
- [ ] 区分 field strength、potential 和 potential energy。
- [ ] 理解 uniform field 和 radial field。
- [ ] 用 Coulomb's law 连接电荷间相互作用。
- [ ] 用 potential gradient 理解 field。

### [[10 Physics/01 Topics/19 Capacitance|Capacitance]]

- [ ] 理解 capacitor 储存 charge 和 energy。
- [ ] 会分析 capacitors in series / parallel。
- [ ] 从 Q-V graph 理解 stored energy。
- [ ] 用 exponential decay 理解 discharging capacitor。
- [ ] 把 RC time constant 与实际电路行为联系起来。

### [[10 Physics/01 Topics/20 Magnetic Fields|Magnetic Fields]]

- [ ] 理解 magnetic field 来自 moving charges 或 magnets。
- [ ] 会判断 current-carrying conductor 和 moving charge 的受力方向。
- [ ] 理解 Hall effect。
- [ ] 把 magnetic force 与 circular motion 联系起来。
- [ ] 理解 magnetic flux、flux linkage、Faraday's law 和 Lenz's law。

### [[10 Physics/01 Topics/21 Alternating Currents|Alternating Currents]]

- [ ] 用 sinusoidal graph 描述 AC。
- [ ] 理解 peak value、r.m.s. value 和 mean power。
- [ ] 理解 half-wave / full-wave rectification。
- [ ] 理解 smoothing capacitor 对输出波形的影响。

## 热、物质与现代物理

### [[10 Physics/01 Topics/14 Temperature|Temperature]]

- [ ] 区分 temperature、thermal energy 和 internal energy。
- [ ] 理解 thermodynamic temperature scale。
- [ ] 用 specific heat capacity 和 latent heat 处理能量转移。
- [ ] 从 heating / cooling curve 读物理过程。

### [[10 Physics/01 Topics/15 Ideal Gases|Ideal Gases]]

- [ ] 用 mole 和 Avogadro constant 连接宏观量与粒子数。
- [ ] 使用 ideal gas equation。
- [ ] 用 kinetic theory 解释 pressure 和 temperature。
- [ ] 理解 r.m.s. speed 和 molecular kinetic energy。

### [[10 Physics/01 Topics/16 Thermodynamics|Thermodynamics]]

- [ ] 理解 internal energy 是状态量。
- [ ] 区分 heat transfer 和 work done。
- [ ] 用 first law of thermodynamics 分析能量流动。
- [ ] 用 p-V graph 理解气体做功。

### [[10 Physics/01 Topics/11 Particle Physics|Particle Physics]]

- [ ] 理解 nuclear atom model。
- [ ] 区分 proton、neutron、electron、quark、lepton、hadron。
- [ ] 用守恒量写 decay equations。
- [ ] 理解 alpha、beta、gamma radiation 的本质。
- [ ] 从粒子模型看 beta decay 和 antiparticles。

### [[10 Physics/01 Topics/22 Quantum Physics|Quantum Physics]]

- [ ] 理解 photon 是能量量子。
- [ ] 用 photoelectric effect 理解光的粒子性。
- [ ] 用 diffraction 理解粒子的波动性。
- [ ] 理解 de Broglie wavelength。
- [ ] 用能级解释 emission 和 absorption spectra。

### [[10 Physics/01 Topics/23 Nuclear Physics|Nuclear Physics]]

- [ ] 用 mass-energy equivalence 理解核能。
- [ ] 理解 mass defect 和 binding energy。
- [ ] 从 binding energy per nucleon graph 理解 fusion 和 fission。
- [ ] 理解 radioactive decay 的随机性和指数规律。
- [ ] 用 half-life 和 decay constant 连接模型与数据。

### [[10 Physics/01 Topics/24 Medical Physics|Medical Physics]]

- [ ] 理解 ultrasound 的产生、反射和 attenuation。
- [ ] 理解 X-ray 产生、contrast 和 attenuation。
- [ ] 理解 CT 如何从多角度图像重建结构。
- [ ] 理解 PET 中 tracer、annihilation 和 gamma photons。
- [ ] 能比较不同医学成像方法的物理依据。

- [ ] 理解 standard candles。

- [ ] 用 redshift 理解宇宙膨胀。
- [ ] 用 Hubble's law 建立宇宙尺度的数量感。
