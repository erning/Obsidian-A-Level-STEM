# 学习地图

这份地图把 9702 当作一条完整的物理自学路线。Syllabus 的价值在于给出边界：它告诉我们一套高中到大学前衔接的物理知识应该覆盖哪些模型、工具和实验思维。学习时不必围绕外部分类转，而应围绕“我能不能解释、计算、建模、验证”来推进。

## 总路线

1. **先学测量和语言**：单位、量纲、误差、矢量、图像和基础数学。
2. **进入经典力学**：运动、力、动量、能量、平衡、材料、圆周运动和引力。
3. **学习波与振动**：波的传播、叠加、干涉、驻波、SHM、共振。
4. **建立场的观念**：电流、电路、电场、磁场、电磁感应和交流电。
5. **理解物质与热**：温度、理想气体、热力学、粒子和原子核。
6. **走向现代物理与应用**：量子、医学成像、天体和宇宙学。
7. **贯穿实验思维**：每个主题都问一句：如果我相信这个模型，怎么测？数据会长什么样？

## 模块地图

| 模块 | 要学什么 | 怎么学 |
|---|---|---|
| 基础工具 | 量、单位、误差、矢量、图像、数学表达 | 每条公式都做单位检查；每张图都说出斜率、截距、面积的物理意义。 |
| 力学 | 运动、力、动量、能量、平衡、材料、圆周运动、引力 | 先画对象和受力，再选守恒量；把“运动图像”和“力学原因”连起来。 |
| 波与振动 | 行波、Doppler、偏振、叠加、干涉、驻波、SHM、共振 | 一边画图一边学；重点理解相位、频率、能量和边界条件。 |
| 电磁学 | 电荷、电流、电路、电场、电容、磁场、电磁感应、交流电 | 把电路看成能量和电荷流动，把场看成空间中每一点的作用规则。 |
| 热与物质 | 温度、热量、气体、内能、粒子、原子核 | 在宏观量和微观模型之间来回翻译。 |
| 现代物理 | 光子、光电效应、物质波、能级、核能 | 接受模型更抽象，但仍坚持“证据是什么、公式怎么用、限制在哪里”。 |
| 应用与宇宙 | 医学成像、恒星、红移、宇宙膨胀 | 把公式放进真实尺度，练习数量级和解释能力。 |
| 实验思维 | 测量、表格、图像、误差、变量控制、线性化 | 不只会算答案，还要会判断一个测量是否可信。 |

## 每个主题的学习循环

每个 topic 都按同一个循环走：

1. **现象**：这个主题想解释什么现象？
2. **模型**：用哪些理想化对象和假设？
3. **量**：核心物理量是什么，单位是什么？
4. **关系**：公式从哪里来，适用条件是什么？
5. **图像**：斜率、截距、面积或曲线形状代表什么？
6. **实验**：怎样测量、验证或反驳这个模型？
7. **迁移**：这个模型能解释哪些日常、工程或宇宙尺度现象？

## 推荐顺序

不追求一次学完所有细节。建议按三轮推进：

1. **第一轮：建立地图**  
   快速过一遍所有主题，知道每个主题解决什么问题。每个主题只写 5-10 条核心理解。

2. **第二轮：深挖模型**  
   回到教材，逐章做 worked examples 和关键练习。每个公式都补适用条件、单位检查和典型误用。

3. **第三轮：做连接**  
   把不同主题连起来，例如 circular motion 与 magnetic force，SHM 与 waves，electric potential 与 gravitational potential，exponential decay 与 capacitor discharge。

## 最重要的学习成果

学完这一套，不只是“知道公式”，而是能做到：

- 看见一个物理情境，能选择合理模型。
- 能把文字、图像、公式、单位和实验数据相互转换。
- 能判断答案数量级是否荒唐。
- 能解释一个公式为什么成立，也知道它什么时候不成立。
- 能设计一个粗略实验来测量某个量，知道主要误差在哪里。
