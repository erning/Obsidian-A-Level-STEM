# Gravitational Fields Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use gravitational field strength as force per unit mass. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Use Newton's law of gravitation for point masses. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Use radial field expressions and compare field strength with potential. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use gravitational potential as work done per unit mass from infinity. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Analyse orbital motion by combining gravity with circular motion. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Calculate field strength, potential, and orbital speed.

## Problem 7

Complete this practice task: Interpret graphs of field strength and potential against distance.

## Problem 8

Complete this practice task: Solve satellite and planetary motion problems.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
