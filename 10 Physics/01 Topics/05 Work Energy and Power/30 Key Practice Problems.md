# Work, Energy and Power Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use work done as force times displacement in the direction of the force. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Apply conservation of energy and efficiency as useful output over total input. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Define power as work done per unit time and use $P = W/t$ and $P = Fv$. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use $\Delta E_p = mg\Delta h$ for gravitational potential energy changes in a uniform gravitational field. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Use $E_k = \frac{1}{2}mv^2$ for kinetic energy. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Solve mechanical energy problems with and without losses.

## Problem 7

Complete this practice task: Use power for steady motion problems.

## Problem 8

Complete this practice task: Compare an energy solution with a force-based solution.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
