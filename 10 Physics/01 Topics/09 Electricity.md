# 09 Electricity

## 教材对应

- 教材：Chapters 8, 10
- 参考范围：9.1 Electric current, 9.2 Potential difference and power, 9.3 Resistance and resistivity

## 需要掌握

- 理解 electric current 是 charge carriers 的流动。
- 理解 charge is quantised。
- 使用 `$Q = It$`。
- 对导体使用 `$I = Anvq$`。
- 定义 potential difference 为每单位电荷转移的能量。
- 使用 `$V = W/Q$`。
- 使用 `$P = VI$`、`$P = I^2R$`、`$P = V^2/R$`。
- 定义 resistance，并使用 `$V = IR$`。
- 画 metallic conductor、semiconductor diode 和 filament lamp 的 I-V characteristics。
- 解释 filament lamp resistance 随温度升高而增加。
- 陈述 Ohm's law。
- 使用 resistivity：`$R = \rho L/A$`。
- 理解 LDR 和 thermistor 的电阻变化趋势。

## 练习方向

- 从 charge-time graph 或 current-time graph 求电荷。
- 用 drift model 连接 current、number density、area 和 drift velocity。
- 从 I-V 图像判断元件类型。
- 计算电阻率或导线长度、面积变化对电阻的影响。
- 处理电功率和能量转移。

## 实验或观察连接

- 搭建 I-V characteristic 实验电路。
- 用 variable resistor 改变电流，记录 p.d. 和 current。
- 比较 metallic conductor、filament lamp、diode、LDR 和 thermistor 的图像。

## 易错点

- I-V 图中 resistance 不一定等于曲线 gradient，要看坐标轴是哪一个。
- Ohm's law 只适用于温度等物理条件恒定时。
- 电流方向与电子 drift direction 相反。
- `$I = Anvq$` 中 `A` 是 cross-sectional area，不是电流表读数。
