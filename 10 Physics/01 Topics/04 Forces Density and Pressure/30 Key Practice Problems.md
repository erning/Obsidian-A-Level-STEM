# Forces, Density and Pressure Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use centre of gravity, moment of a force, couple, and torque of a couple. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Apply the principle of moments and the condition of no resultant force and no resultant torque for equilibrium. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Define and use density and pressure. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Derive and use hydrostatic pressure $\Delta p = \rho g \Delta h$. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Explain upthrust from pressure difference and use $F = \rho gV$ as Archimedes' principle. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Solve moment and torque problems.

## Problem 7

Complete this practice task: Use vector triangles for coplanar forces in equilibrium.

## Problem 8

Complete this practice task: Calculate hydrostatic pressure and upthrust.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
