# Electric Fields Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use electric field strength as force per unit positive charge. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Represent electric fields with field lines and distinguish uniform and radial fields. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Use force on a charge in a uniform electric field. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use Coulomb's law for point charges and field strength due to a point charge. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Use electric potential and potential gradient to connect energy and field strength. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Calculate electric force, field strength, potential, and work done.

## Problem 7

Complete this practice task: Interpret field-line and equipotential diagrams.

## Problem 8

Complete this practice task: Analyse charged-particle motion in uniform fields.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
