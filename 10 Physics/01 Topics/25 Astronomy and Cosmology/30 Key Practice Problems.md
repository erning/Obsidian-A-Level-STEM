# Astronomy and Cosmology Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use luminosity and radiant flux intensity to relate brightness and distance. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Use standard candles as distance indicators. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Use Wien's displacement law and Stefan-Boltzmann law to estimate stellar temperature, radius, and luminosity. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use redshift to infer recession speed for distant galaxies. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Use Hubble's law as evidence for the expanding universe and Big Bang model. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Calculate distances from luminosity and flux.

## Problem 7

Complete this practice task: Estimate stellar radii and temperatures.

## Problem 8

Complete this practice task: Use redshift and Hubble's law for cosmological scale estimates.

## Extension
