# Magnetic Fields Review Checklist

Use this checklist after the practice problems. For every calculation, check
the angle in the formula and the direction rule separately.

## Ideas to Recall

- [ ] A magnetic field is a force field produced by moving charges or
      permanent magnets.
- [ ] Field lines show direction, and closer spacing means stronger field.
- [ ] The right-hand grip rule gives magnetic field direction around a current.
- [ ] A current-carrying conductor in a magnetic field may experience a force.
- [ ] Magnetic flux density is force per unit current per unit length for a
      wire at right angles to the field.
- [ ] Fleming's left-hand rule gives the direction of the motor-effect force.
- [ ] A moving charge in a magnetic field experiences a force only if it has a
      velocity component perpendicular to the field.
- [ ] Magnetic force is perpendicular to the motion, so it changes direction
      without changing speed.
- [ ] Hall voltage comes from charge separation caused by magnetic force on
      moving charge carriers.
- [ ] A Hall probe measures magnetic flux density using a calibrated Hall
      voltage.
- [ ] A charged particle entering a uniform magnetic field at right angles
      follows a circular path.
- [ ] Crossed electric and magnetic fields can select particles of one speed.
- [ ] Currents in the same direction attract; currents in opposite directions
      repel.
- [ ] A ferrous core increases the magnetic field in a solenoid.
- [ ] Magnetic flux is $\Phi=BA$ when the field is perpendicular to the area.
- [ ] Flux linkage is $N\Phi$.
- [ ] Faraday's law links induced e.m.f. to the rate of change of flux
      linkage.
- [ ] Lenz's law says the induced effect opposes the change producing it.

## Skills to Perform

- [ ] Sketch field patterns for a straight wire, flat circular coil, and long
      solenoid.
- [ ] Use $F=BIL\sin\theta$ for a current-carrying conductor.
- [ ] Define and calculate magnetic flux density in tesla.
- [ ] Use $F=BQv\sin\theta$ for a moving charge.
- [ ] Derive and use $V_H=\frac{BI}{ntq}$ for Hall voltage.
- [ ] Use $r=\frac{mv}{BQ}$ for circular motion in a uniform magnetic field.
- [ ] Use $v=\frac{E}{B}$ for velocity selection.
- [ ] Explain attraction and repulsion between current-carrying conductors.
- [ ] Calculate magnetic flux and flux linkage.
- [ ] Use $|\mathcal{E}|=\left|\frac{\Delta(N\Phi)}{\Delta t}\right|$.
- [ ] Use $\mathcal{E}=BLv$ for a straight conductor moving perpendicular to a
      magnetic field.
- [ ] Apply Lenz's law as an energy and direction check.

## Common Errors to Avoid

- [ ] Applying magnetic force to a stationary charge.
- [ ] Forgetting the $\sin\theta$ factor in force equations.
- [ ] Using the angle to the force instead of the angle to the magnetic field.
- [ ] Forgetting to reverse the force direction for a negative charge.
- [ ] Treating magnetic force as if it increases the speed of a charged
      particle.
- [ ] Confusing magnetic flux density $B$ with magnetic flux $\Phi$.
- [ ] Using $\Phi=BA$ when the field is not perpendicular to the area.
- [ ] Forgetting the number of turns in flux linkage.
- [ ] Saying Lenz's law opposes the magnetic field rather than the change in
      flux linkage.
- [ ] Treating Hall voltage as a supply voltage instead of a charge-separation
      effect.

## Ready to Move On When

- [ ] I can choose the correct direction rule before starting a calculation.
- [ ] I can calculate forces on wires and moving charges with the correct
      angle.
- [ ] I can explain Hall voltage and use it to calculate magnetic flux density
      or Hall p.d.
- [ ] I can analyse circular charged-particle motion and velocity selection.
- [ ] I can sketch current-produced magnetic fields and explain wire forces.
- [ ] I can calculate flux, flux linkage, and induced e.m.f.
- [ ] I can use Lenz's law to identify what change is being opposed.
- [ ] I can connect this topic to [[10 Physics/01 Topics/12 Motion in a Circle/00 Overview|Motion in a Circle]],
