# Kinematics Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Define and use distance, displacement, speed, velocity, and acceleration. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Read displacement from area under a velocity-time graph, velocity from the gradient of a displacement-time graph, and acceleration from the gradient of a velocity-time graph. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Derive and use equations for uniformly accelerated motion in a straight line. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Describe free fall in a uniform gravitational field without air resistance. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Analyse projectile motion as uniform velocity in one direction and uniform acceleration in a perpendicular direction. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Translate between words, graphs, and equations of motion.

## Problem 7

Complete this practice task: Solve free-fall and projectile problems with clear sign conventions.

## Problem 8

Complete this practice task: Use graph gradients and areas as physical quantities.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
