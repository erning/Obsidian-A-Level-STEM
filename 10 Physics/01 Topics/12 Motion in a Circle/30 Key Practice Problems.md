# Motion in a Circle Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use radian measure, angular displacement, angular speed, period, and frequency. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Use $v = r\omega$ for uniform circular motion. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Use centripetal acceleration and centripetal force directed towards the centre. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Identify the real force or resultant force that supplies the centripetal role. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Connect circular motion to orbits, charged-particle motion, and oscillations. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Solve vertical and horizontal circular-motion contexts.

## Problem 7

Complete this practice task: Relate angular speed to period and frequency.

## Problem 8

Complete this practice task: Use circular motion in gravitational and magnetic-field problems.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
