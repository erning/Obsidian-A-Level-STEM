# Deformation of Solids Review Checklist

Use this after the worked examples and practice problems. Any unchecked item is
a cue to return to the lecture notes and redo one relevant graph or
calculation.

## Core Ideas to Recall

- [ ] A tensile force stretches a material; a compressive force shortens it.
- [ ] Load is the applied force, and extension is the increase in length.
- [ ] Hooke's law states that extension is directly proportional to load,
  provided the limit of proportionality is not exceeded.
- [ ] The spring constant is $k=F/x$ and has unit
  $\mathrm{N\ m^{-1}}$.
- [ ] The gradient of a force-extension graph in the straight-line region gives
  the spring constant.
- [ ] Elastic deformation is reversible; plastic deformation is permanent.
- [ ] The elastic limit and the limit of proportionality are different ideas.
- [ ] Stress is $\sigma=F/A$ and has unit $\mathrm{Pa}$.
- [ ] Strain is $\varepsilon=x/L$ and has no unit.
- [ ] Young modulus is $E=\sigma/\varepsilon$ and is the gradient of the
  initial linear stress-strain graph.
- [ ] Work done in deformation is the area under the force-extension graph.
- [ ] Within the limit of proportionality,
  $E_p=\frac{1}{2}Fx=\frac{1}{2}kx^2$.

## Skills to Perform

- [ ] Convert between $\mathrm{mm}$, $\mathrm{cm}$, and $\mathrm{m}$ before
  using deformation formulae.
- [ ] Calculate a spring constant from a load and extension.
- [ ] Use a force-extension graph to find both stiffness and work done.
- [ ] Calculate stress from force and cross-sectional area.
- [ ] Calculate strain from extension and original length.
- [ ] Calculate Young modulus from stress and strain, or from
  $E=FL/(Ax)$.
- [ ] Use $A=\pi d^2/4$ when a wire diameter is given.
- [ ] Describe a Young modulus experiment for a metal wire.
- [ ] Interpret loading and unloading curves, including permanent extension and
  energy dissipated.

## Common Errors to Avoid

- [ ] Confusing extension with strain.
- [ ] Confusing spring constant $k$ with Young modulus $E$.
- [ ] Using Hooke's law beyond the proportional region.
- [ ] Assuming the elastic limit is automatically the same as the limit of
  proportionality.
- [ ] Using diameter instead of cross-sectional area in stress.
- [ ] Using final length instead of original length in strain.
- [ ] Forgetting that strain has no unit.
- [ ] Forgetting that graph area, not graph gradient, gives work done.
- [ ] Using $E_p=\frac{1}{2}kx^2$ for a non-linear or plastically deformed
  region.

## Quick Self-Checks

- [ ] I can explain why a long, thin wire is useful in a Young modulus
  experiment.
- [ ] I can state which measurements are needed to calculate $E=FL/(Ax)$.
- [ ] I can find the Young modulus from a stress-strain graph.
- [ ] I can decide from an unloading graph whether deformation is elastic or
  plastic.
- [ ] I can explain why the energy stored in a Hooke's law spring is
  $\frac{1}{2}kx^2$, not $kx^2$.
