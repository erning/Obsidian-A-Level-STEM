# Alternating Currents Review Checklist

Use this checklist after the practice problems. If a calculation is wrong,
first check whether the value used was peak, peak-to-peak, r.m.s., maximum, or
mean.

## Ideas to Recall

- [ ] Alternating current changes direction periodically.
- [ ] The standard waveform in this topic is sinusoidal.
- [ ] Peak value is the maximum magnitude of current or voltage.
- [ ] Period is the time for one complete cycle.
- [ ] Frequency is the number of cycles per second, with $f=1/T$.
- [ ] Angular frequency is related to frequency by $\omega=2\pi f$.
- [ ] A sinusoidal quantity can be represented by $x=x_0\sin\omega t$.
- [ ] The r.m.s. value gives the equivalent steady d.c. value for the same
      mean power in the same resistor.
- [ ] For a sinusoidal waveform,
      $I_{\text{r.m.s.}}=\frac{I_0}{\sqrt{2}}$ and
      $V_{\text{r.m.s.}}=\frac{V_0}{\sqrt{2}}$.
- [ ] In a resistive load, the mean power is half the maximum power for a
      sinusoidal alternating current.
- [ ] Rectification makes current through the load pass in one direction only.
- [ ] Half-wave rectification uses one diode and removes one half-cycle.
- [ ] Full-wave bridge rectification uses four diodes and uses both
      half-cycles.
- [ ] A smoothing capacitor charges near peaks and discharges through the load
      between peaks.
- [ ] Smoothing reduces ripple but does not remove it completely.

## Skills to Perform

- [ ] Read peak voltage and period from an oscilloscope trace.
- [ ] Convert between period, frequency, and angular frequency.
- [ ] Use equations of the form $x=x_0\sin\omega t$.
- [ ] Convert between peak and r.m.s. values for a sinusoidal waveform.
- [ ] Calculate maximum instantaneous power and mean power in a resistor.
- [ ] Use r.m.s. values in $P=VI$, $P=I^2R$, and $P=V^2/R$.
- [ ] Sketch half-wave and full-wave rectified outputs.
- [ ] Explain the four-diode bridge rectifier qualitatively.
- [ ] Analyse how capacitance and load resistance affect smoothing.

## Common Errors to Avoid

- [ ] Using peak voltage directly in a mean power formula.
- [ ] Confusing peak voltage with peak-to-peak voltage.
- [ ] Confusing frequency $f$ with angular frequency $\omega$.
- [ ] Forgetting to convert milliseconds to seconds.
- [ ] Treating the r.m.s. formulae as valid for every possible waveform.
- [ ] Saying rectification alone produces a steady d.c. output.
- [ ] Drawing full-wave rectified output with alternating positive and negative
      pulses.
- [ ] Saying a smoothing capacitor removes ripple completely.
- [ ] Forgetting that heavier load current increases ripple.

## Ready to Move On When

- [ ] I can read a sinusoidal trace and identify peak, peak-to-peak, period,
      frequency, and r.m.s. values.
- [ ] I can use $x=x_0\sin\omega t$ with time in seconds and angle in radians.
- [ ] I can decide whether a power question needs peak or r.m.s. values.
- [ ] I can explain why mean power in a resistive load is half the maximum
      power for a sinusoidal alternating current.
- [ ] I can sketch and compare half-wave and full-wave rectification.
- [ ] I can explain capacitor smoothing using charge, discharge, and ripple.
- [ ] I can connect this topic to [Capacitance](../19%20Capacitance/00%20Overview.md)
      and [Magnetic Fields](../20%20Magnetic%20Fields/00%20Overview.md).
