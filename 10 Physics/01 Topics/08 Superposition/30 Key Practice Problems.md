# Superposition Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use the principle of superposition for waves meeting at a point. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Describe stationary waves using nodes, antinodes, phase, and harmonic modes. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Understand diffraction at a slit or edge and the role of aperture size relative to wavelength. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use path difference, phase difference, and coherence for two-source interference. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Use diffraction grating relationships to connect spectra with wavelength. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Find node and antinode spacing.

## Problem 7

Complete this practice task: Calculate fringe spacing and grating angles.

## Problem 8

Complete this practice task: Explain how changing wavelength or slit separation changes the pattern.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
