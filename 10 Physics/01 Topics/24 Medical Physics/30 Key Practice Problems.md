# Medical Physics Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Explain ultrasound production, reflection, acoustic impedance, attenuation, and echo timing. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Explain X-ray production, attenuation, image contrast, and computed tomography principles. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Describe PET scanning using tracers, positron emission, annihilation, and gamma-ray detection. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Compare imaging methods by resolution, contrast mechanism, penetration, risk, and information produced. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Use exponential attenuation models where appropriate. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Calculate echo distances and attenuation.

## Problem 7

Complete this practice task: Compare X-ray and ultrasound image formation.

## Problem 8

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
