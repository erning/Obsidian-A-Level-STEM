# Physical Quantities and Units Review Checklist

Use this after working through [[10 Physics/01 Topics/01 Physical Quantities and Units/10 Lecture Notes|Lecture Notes]], [[10 Physics/01 Topics/01 Physical Quantities and Units/20 Worked Examples|Worked Examples]], and [[10 Physics/01 Topics/01 Physical Quantities and Units/30 Key Practice Problems|Key Practice Problems]].

## Core Ideas

- [ ] I can explain why a physical quantity needs both a numerical magnitude and a unit.
- [ ] I can make order-of-magnitude estimates and reject values that are physically unreasonable.
- [ ] I can recall the SI base units for mass, length, time, current, and temperature.
- [ ] I can use prefixes from pico to tera without losing powers of ten.
- [ ] I can explain why cubed and squared units change the prefix factor.
- [ ] I can distinguish random errors, systematic errors, and zero errors.
- [ ] I can distinguish precision from accuracy using repeated readings and true-value language.
- [ ] I can distinguish scalars from vectors and give syllabus-based examples of each.

## Calculation Skills

- [ ] I can convert measurements such as $\mathrm{cm^3}$, $\mu\mathrm{s}$, and $\mathrm{G}\Omega$ into SI units.
- [ ] I can derive units such as $\mathrm{N}$, $\mathrm{Pa}$, $\mathrm{C}$, and $\mathrm{V}$ from defining equations.
- [ ] I can check whether an equation is homogeneous using SI base units.
- [ ] I can find absolute, fractional, and percentage uncertainty.
- [ ] I can combine absolute uncertainties for sums and differences.
- [ ] I can combine percentage uncertainties for products, quotients, and powers.
- [ ] I can correct a measurement for a zero error before quoting the final value.
- [ ] I can add and subtract coplanar vectors using a diagram or components.
- [ ] I can resolve a vector into perpendicular components using the angle with the chosen direction.
- [ ] I can state a vector answer with both magnitude and direction.

## Common Errors to Avoid

- [ ] Writing a number without a unit.
- [ ] Treating $\mathrm{kg}$ and $\mathrm{g}$ as interchangeable in SI calculations.
- [ ] Converting $1\ \mathrm{cm^3}$ as $10^{-2}\ \mathrm{m^3}$ instead of $10^{-6}\ \mathrm{m^3}$.
- [ ] Using a unit check as proof that an equation is correct.
- [ ] Rounding the measured value more finely than the uncertainty justifies.
- [ ] Trying to reduce a systematic error by taking more repeated readings.
- [ ] Subtracting uncertainties because one measured value is subtracted from another.
- [ ] Adding vector magnitudes when the vectors point in different directions.
- [ ] Giving an acceleration or force without a direction when direction matters.

## Ready to Connect

You are ready to connect this topic to [[10 Physics/01 Topics/02 Kinematics/00 Overview|Kinematics]] when you can:

- convert all distances, times, velocities, and accelerations to consistent SI units;
- tell the difference between distance and displacement, and between speed and velocity;
- use vector components to describe motion in perpendicular directions;
- check the unit of each term in a motion equation.

You are ready to connect this topic to [[10 Physics/02 Experimental Thinking/Measurement and Data Presentation/00 Overview|Measurement and Data Presentation]] when you can:

- choose a sensible uncertainty for a single reading or repeated readings;
- recognise whether a measurement problem is random or systematic;
- calculate percentage uncertainty in a derived quantity;
- quote a final measured result with a suitable unit and uncertainty.
