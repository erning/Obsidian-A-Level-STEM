# Physical Quantities and Units Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use physical quantities as numerical magnitude plus unit, and make reasonable order-of-magnitude estimates. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Use SI base units, derived units, prefixes, and dimensional homogeneity checks. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Distinguish systematic errors, random errors, precision, accuracy, and simple uncertainty propagation. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Distinguish scalar and vector quantities, then add, subtract, and resolve coplanar vectors. State assumptions, choose a method, and check the result.

## Problem 5

Complete this practice task: Convert derived units into base units.

## Problem 6

Complete this practice task: Estimate percentage uncertainties in derived quantities.

## Problem 7

Complete this practice task: Resolve vectors into perpendicular components and reconstruct the resultant.

## Problem 8

Solve a representative Physical Quantities and Units problem and check it using a second representation.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
