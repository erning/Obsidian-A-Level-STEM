# Capacitance Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Define capacitance using $C = Q/V$. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Use equivalent capacitance for capacitors in series and in parallel. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Use area under a charge-potential graph to find energy stored in a capacitor. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use $W = \frac{1}{2}QV = \frac{1}{2}CV^2$. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Analyse capacitor discharge graphs for current, charge, and potential difference. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Calculate equivalent capacitance in mixed networks.

## Problem 7

Complete this practice task: Find stored energy from formulae and graphs.

## Problem 8

Complete this practice task: Extract time constant from discharge data.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
