# Dynamics Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use mass as the property that resists change in motion, and use $F = ma$ with acceleration in the direction of resultant force. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Define linear momentum as mass times velocity and force as rate of change of momentum. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: State and apply Newton's laws of motion. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Describe frictional, viscous, and drag forces qualitatively, including terminal velocity. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Apply conservation of momentum to elastic and inelastic interactions in one and two dimensions. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Use $F = ma$ with components.

## Problem 7

Complete this practice task: Calculate impulse and changes in momentum.

## Problem 8

Complete this practice task: Solve collision problems and interpret kinetic energy changes.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
