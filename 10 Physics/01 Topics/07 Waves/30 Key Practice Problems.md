# Waves Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Use displacement, amplitude, phase difference, period, frequency, wavelength, and speed. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Use the wave equation $v = f\lambda$ and intensity relationships for progressive waves. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Distinguish transverse and longitudinal waves and understand polarisation as evidence for transverse waves. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Describe Doppler effect for sound waves. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Know the main regions and order of the electromagnetic spectrum. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Calculate wavelength, frequency, speed, period, and phase difference.

## Problem 7

Complete this practice task: Interpret CRO-style wave traces.

## Problem 8

Complete this practice task: Explain Doppler shifts qualitatively.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
