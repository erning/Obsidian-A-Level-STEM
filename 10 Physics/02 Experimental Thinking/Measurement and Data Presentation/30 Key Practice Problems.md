# Measurement and Data Presentation Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Choose measuring instruments with suitable range and precision. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Record raw data, processed data, units, and uncertainties clearly. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Use significant figures consistently with the measurement precision. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Plot graphs with labelled axes, sensible scales, and appropriate best-fit lines. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Interpret gradient, intercept, area, and uncertainty in graphical work. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Build tables from raw data and calculated columns.

## Problem 7

Complete this practice task: Draw best-fit lines and estimate gradients with large triangles.

## Problem 8

Complete this practice task: Use uncertainty bars where they help judge the fit.

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
