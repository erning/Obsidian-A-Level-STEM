# Experimental Design and Data Analysis Key Practice Problems

Work these without the solution note first. The aim is active recall, method selection, and clean checking.

## Problem 1

Use this idea in a complete solution: Define independent, dependent, and control variables. State assumptions, choose a method, and check the result.

## Problem 2

Use this idea in a complete solution: Choose a method that makes the relationship measurable and repeatable. State assumptions, choose a method, and check the result.

## Problem 3

Use this idea in a complete solution: Identify major uncertainty sources and practical limitations. State assumptions, choose a method, and check the result.

## Problem 4

Use this idea in a complete solution: Use linearisation when it gives a clearer test of a relationship. State assumptions, choose a method, and check the result.

## Problem 5

Use this idea in a complete solution: Draw conclusions from data, not from the formula alone. State assumptions, choose a method, and check the result.

## Problem 6

Complete this practice task: Convert relationships into straight-line graph forms.

## Problem 7

Complete this practice task: Propose control variables for mechanics, waves, circuits, thermal physics, and radioactive decay contexts.

## Problem 8

## Extension

Choose one problem above and change a condition. Explain how the method changes and which parts of the original solution still survive.
