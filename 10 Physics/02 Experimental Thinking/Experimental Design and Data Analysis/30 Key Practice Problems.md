# Experimental Design and Data Analysis Key Practice Problems

Work these without looking at the solutions first. For every plan, write what
you will vary, what you will measure, what you will keep constant, what graph
you will plot, and how the graph gives the required constant.

## A. Planning and Variables

1. For an experiment testing how the period $T$ of a pendulum depends on length
   $l$, identify the independent variable, dependent variable, and three
   control variables.

2. For each control variable in question 1, state how it would be kept
   constant.

3. Write a concise plan to test the relationship
   $T=2\pi\sqrt{l/g}$ for a pendulum. Include apparatus, range, repeats,
   graph, and how $g$ is found.

4. A circuit experiment tests whether a fixed resistor obeys $V=IR$. State the
   independent variable, dependent variable, two control variables, and the
   graph to plot.

5. State a hazard and matching safety precaution for an experiment using a
   loaded spring or stretched wire.

6. Explain why using a wide range of independent-variable values and at least
   six readings improves the investigation.

## B. Linearisation and Data Analysis

7. A relationship has the form $y=ax^n$. State what graph should be plotted,
   and state how $a$ and $n$ are obtained.

8. A relationship has the form $y=ae^{kx}$. State what graph should be plotted,
   and state how $a$ and $k$ are obtained.

9. For a pendulum, a graph of $T^2$ against $l$ has best-fit gradient
   $4.05\ \mathrm{s^2\ m^{-1}}$. Find $g$.

10. The worst acceptable line for the graph in question 9 has gradient
    $4.20\ \mathrm{s^2\ m^{-1}}$. Find the uncertainty in $g$ and quote the
    result.

11. A straight-line graph has best-fit gradient $3.60$ and worst acceptable
    gradient $3.25$, in the same units. Find the absolute uncertainty in the
    gradient.

12. A line of best fit has gradient $1.80$ and passes through $(2.0, 5.1)$.
    Find the intercept.

13. A logarithmic table uses $L=76.5\ \mathrm{cm}$. State a suitable table
    entry for $\lg(L/\mathrm{cm})$ and explain the number of decimal places.

14. The voltage in a capacitor discharge follows
    $V=V_0e^{-t/RC}$. State what graph should be plotted and how $RC$ is found
    from the gradient.

## C. Conclusions, Evaluation, and Instruments

15. An experiment gives $g=(9.7\pm0.3)\ \mathrm{m\ s^{-2}}$. The accepted
    value is $9.81\ \mathrm{m\ s^{-2}}$. State whether the result is
    consistent with the accepted value.

16. Why should a conclusion say "consistent with the model" rather than "proves
    the model"?

17. A student writes, "human error affected the timing." Rewrite this as a
    specific limitation and matching improvement for a pendulum experiment.

18. A student suggests "use a more accurate ruler" for an experiment where the
    main scatter comes from hand timing. Explain why this is a weak
    improvement.

19. A graph has points that curve systematically away from a straight best-fit
    line. What does this suggest about the model or analysis?

20. State a hazard and matching safety precaution for an experiment with a hot
    liquid.

21. In a radioactive decay experiment, state two variables or conditions that
    should be controlled or corrected for when analysing count rate.

22. Describe how an oscilloscope can be used to measure a time interval or
    frequency.

23. Describe how light gates connected to a data logger can measure speed.

24. Explain why an improvement that was already possible with the given
    apparatus and instructions may not receive credit.
