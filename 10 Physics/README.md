# CAIE 物理 9702

这个目录以 CAIE 物理 9702 作为主要学习框架和教材线索来学习物理。我们借 9702 限定知识范围、安排主题顺序、连接大纲和教材，但目标不是拿分，而是建立一套扎实的物理理解：会看现象、会建模型、会用数学表达、会从实验和数据中判断一个想法是否靠谱。
