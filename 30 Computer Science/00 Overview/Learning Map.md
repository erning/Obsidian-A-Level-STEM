# Learning Map

This map follows CAIE Computer Science 9618. The syllabus is split into AS Level (sections 1-12) and A Level (sections 13-20), but the better route follows conceptual dependency. Learn a layer of abstraction, then see how the next layer builds on it.

## Route

1. Data representation: binary, denary, hexadecimal, BCD, two's complement, text, images, sound, and compression.
2. Hardware: input/output, memory types, logic gates, and logic circuits.
3. Processor fundamentals: Von Neumann model, registers, buses, fetch-execute cycle, assembly language, and bit manipulation.
4. System software: operating systems, language translators, and IDE features.
5. Security, ethics, and integrity: data security, data integrity, and ethics and ownership.
6. Communication: networks, topologies, the internet, IP addressing, DNS, and cloud computing.
7. Databases: relational model, normalisation, DBMS, DDL, and DML.
8. Algorithms and data structures: computational thinking, pseudocode, flowcharts, arrays, files, and abstract data types.
9. Programming and software development: constructs, procedures, functions, the development life cycle, structure charts, testing, and maintenance.
10. Advanced theory: user-defined data types, file organisation, floating-point numbers, internet protocols, RISC/CISC, parallel processing, virtual machines, Boolean algebra, OS internals, translation, encryption, and artificial intelligence.
11. Advanced problem-solving and programming: searching and sorting algorithms, ADT operations, recursion, complexity, programming paradigms, file processing, and exception handling.

## Learning Questions

Use these questions for every topic:

- What layer of abstraction is this topic on (data, hardware, logic, system, network, algorithm, code)?
- How is the data represented, stored, or moved at this layer?
- What structure, algorithm, or protocol does the work?
- How can the behaviour be traced, tested, or verified?
- What can go wrong: overflow, error, collision, deadlock, or logic fault?
- How does this connect to the layer above and the layer below?

## Three Passes

First pass: build the layers. Learn what each topic does and what it depends on.

Second pass: build methods. Trace algorithms by hand, draw circuits and diagrams, and write pseudocode against the official guide.

Third pass: build connections. Link data representation to processor operations, networks to protocols, algorithms to code, and AS topics to their A Level extensions.
