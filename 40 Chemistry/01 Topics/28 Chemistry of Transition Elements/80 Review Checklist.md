# 28 Chemistry of Transition Elements - Review Checklist

## Definitions and Electronic Structure

- [ ] I can define a transition element as a d-block element that forms one or more stable ions with incomplete d orbitals.
- [ ] I can explain why the definition depends on ions, not only atoms.
- [ ] I can describe the shapes of $3d_{xy}$ and $3d_{z^2}$ orbitals.
- [ ] I can explain variable oxidation states using the similar energies of the $3d$ and $4s$ subshells.

## Characteristic Properties

- [ ] I can state that transition elements have variable oxidation states.
- [ ] I can explain catalytic behaviour using more than one stable oxidation state.
- [ ] I can explain catalytic behaviour using accessible vacant d orbitals and dative bonding.
- [ ] I can explain why transition elements form complex ions.
- [ ] I can explain why transition element compounds can be coloured.

## Complex Ions and Ligands

- [ ] I can define ligand.
- [ ] I can define complex.
- [ ] I can identify monodentate ligands such as $\ce{H2O}$, $\ce{NH3}$, $\ce{Cl-}$, and $\ce{CN-}$.
- [ ] I can identify bidentate ligands such as en and $\ce{C2O4^2-}$.
- [ ] I can identify $\ce{EDTA^4-}$ as polydentate.
- [ ] I can state and use coordination number correctly.
- [ ] I can predict the formula and charge of a complex ion.
- [ ] I can recognise linear, square planar, tetrahedral, and octahedral shapes.

## Ligand Exchange and Observations

- [ ] I can write ligand exchange equations for $\ce{[Cu(H2O)6]^2+}$ with $\ce{NH3}$ and $\ce{Cl-}$.
- [ ] I can write ligand exchange equations for $\ce{[Co(H2O)6]^2+}$ with $\ce{Cl-}$.
- [ ] I can connect copper(II) and cobalt(II) colour changes to ligand exchange and changes in $\Delta E$.
- [ ] I can describe hydroxide precipitate formation with aqueous copper(II) and cobalt(II) ions.

## Redox

- [ ] I can use $E^\circ$ values to predict whether a transition element redox reaction is feasible.
- [ ] I can write the acidified $\ce{MnO4-}$ half-equation.
- [ ] I can write and use the equation for $\ce{MnO4-}$ with $\ce{Fe^{2+}}$ in acid.
- [ ] I can write and use the equation for $\ce{MnO4-}$ with $\ce{C2O4^2-}$ in acid.
- [ ] I can write and use the equation for $\ce{Cu^{2+}}$ with $\ce{I-}$.

## Colour

- [ ] I can define degenerate and non-degenerate d orbitals.
- [ ] I can describe d-orbital splitting in octahedral complexes.
- [ ] I can describe d-orbital splitting in tetrahedral complexes.
- [ ] I can use $\Delta E = h\nu$ to link energy gap and absorbed frequency.
- [ ] I can explain that the observed colour is complementary to the absorbed colour.
- [ ] I can explain how different ligands change $\Delta E$ and colour.

## Stereoisomerism and Polarity

- [ ] I can identify cis/trans isomerism in square planar complexes such as $\ce{[Pt(NH3)2Cl2]}$.
- [ ] I can identify cis/trans isomerism in octahedral complexes such as $\ce{[Co(NH3)4(H2O)2]^2+}$.
- [ ] I can recognise optical isomerism in complexes with bidentate ligands such as $\ce{[Ni(en)3]^2+}$.
- [ ] I can use shape and symmetry to deduce whether dipoles cancel.

## Stability Constants

- [ ] I can define $K_\mathrm{stab}$.
- [ ] I can write a $K_\mathrm{stab}$ expression from a formation equilibrium.
- [ ] I remember not to include water in $K_\mathrm{stab}$ expressions.
- [ ] I can perform calculations using $K_\mathrm{stab}$.
- [ ] I can explain ligand exchange using relative $K_\mathrm{stab}$ values.
