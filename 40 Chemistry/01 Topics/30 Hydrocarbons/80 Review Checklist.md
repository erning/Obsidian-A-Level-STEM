# 30 Hydrocarbons - Review Checklist

## Source Scope

- [ ] I can state that this topic follows CAIE Chemistry 9701 Topic 30.1 Arenes.
- [ ] I can connect A Level arenes to AS alkanes and alkenes.

## Arene Bonding and Mechanism

- [ ] I can explain aromatic stabilisation using a delocalised $\pi$ system.
- [ ] I can describe electrophilic substitution in benzene.
- [ ] I can explain why substitution predominates over addition.

## Reactions and Conditions

- [ ] I know ring halogenation conditions using $\ce{Cl2}$ or $\ce{Br2}$ with $\ce{AlCl3}$ or $\ce{AlBr3}$.
- [ ] I know nitration conditions: concentrated $\ce{HNO3}$, concentrated $\ce{H2SO4}$, $25$ to $60\,^\circ\mathrm{C}$.
- [ ] I know Friedel-Crafts alkylation using $\ce{CH3Cl}$, $\ce{AlCl3}$ and heat.
- [ ] I know Friedel-Crafts acylation using $\ce{CH3COCl}$, $\ce{AlCl3}$ and heat.
- [ ] I know side-chain oxidation using hot alkaline $\ce{KMnO4}$ followed by dilute acid.
- [ ] I know hydrogenation using $\ce{H2}$ and Pt or Ni catalyst with heat.

## Prediction Skills

- [ ] I can decide whether halogenation occurs in the ring or side-chain from conditions.
- [ ] I can use the directing effects of $\ce{-NH2}$, $\ce{-OH}$, $\ce{-R}$, $\ce{-NO2}$, $\ce{-COOH}$ and $\ce{-COR}$.
- [ ] I can predict simple products from benzene and methylbenzene reactions.

## Ready to Move On

- [ ] I can use arene halogenation to support Topic 31 Halogen Compounds.
- [ ] I can use side-chain oxidation to support Topic 33 Carboxylic Acids and Derivatives.
- [ ] I can use electrophilic substitution language in synthesis and analysis problems.
