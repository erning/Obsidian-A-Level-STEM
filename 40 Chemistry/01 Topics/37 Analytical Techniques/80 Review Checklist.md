# 37 Analytical Techniques - Review Checklist

## Source Scope

- [ ] I can state that this topic follows CAIE Chemistry 9701 Topic 37.
- [ ] I can connect A Level chromatography and NMR to AS mass spectrometry and infrared evidence.

## Thin-Layer Chromatography

- [ ] I can define stationary phase, mobile phase, baseline, solvent front and $R_f$ value.
- [ ] I can calculate $R_f$ values.
- [ ] I can explain $R_f$ differences using stationary-phase interaction and mobile-phase solubility.
- [ ] I can use TLC evidence for purity and identification.

## Gas / Liquid Chromatography

- [ ] I can define stationary phase, mobile phase and retention time.
- [ ] I can interpret percentage composition from peak areas.
- [ ] I can explain retention time using interaction with the stationary phase.

## Carbon-13 NMR

- [ ] I can count carbon environments in simple molecules.
- [ ] I can use symmetry when predicting peak numbers.
- [ ] I avoid using carbon-13 peak height as a carbon ratio.

## Proton NMR

- [ ] I can use chemical shift values to identify proton environments.
- [ ] I can use relative peak areas as proton ratios.
- [ ] I can apply the $n+1$ rule for singlets, doublets, triplets, quartets and multiplets.
- [ ] I can explain the use of TMS.
- [ ] I can explain the need for deuterated solvents.
- [ ] I can identify $\ce{O-H}$ and $\ce{N-H}$ protons by $\ce{D2O}$ exchange.

## Ready to Combine Evidence

- [ ] I can combine chromatography and NMR evidence to narrow possible structures.
- [ ] I can use analytical data to support organic synthesis conclusions.
