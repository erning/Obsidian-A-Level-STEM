# 37 Analytical Techniques

## Core Idea

A Level analytical techniques use chromatography and NMR spectroscopy to separate, identify, and confirm organic substances. The topic links observable data to structure, composition, and purity.

## Source Alignment

- Primary syllabus source: CAIE Chemistry 9701, A Level subject content.
- 37 Analytical techniques
- 37.1 Thin-layer chromatography
- 37.2 Gas / liquid chromatography
- 37.3 Carbon-13 NMR spectroscopy
- 37.4 Proton (1H) NMR spectroscopy

## What to Learn

- Use stationary phase, mobile phase, baseline, solvent front, and $R_f$ value in thin-layer chromatography.
- Interpret gas / liquid chromatograms using retention time and percentage composition.
- Use carbon-13 NMR spectra to identify carbon environments and possible structures.
- Use proton NMR chemical shift, relative peak area, splitting pattern, TMS, deuterated solvents, and D2O exchange.
- Combine analytical evidence to narrow possible structures.

## How to Study

- For chromatography, separate movement caused by solubility in the mobile phase from interaction with the stationary phase.
- For NMR, count chemically different environments before reading detailed data.
- Use relative peak areas only where the technique provides them.
- Build structure suggestions gradually from all evidence, not from a single peak.

## Practice Directions

- Calculate and interpret $R_f$ values from TLC diagrams.
- Interpret retention times and percentage composition from gas / liquid chromatograms.
- Predict the number of carbon-13 NMR peaks for simple molecules.
- Interpret proton NMR spectra using chemical shifts, integration, and the $n + 1$ rule.

## Common Traps

- Confusing $R_f$ value with retention time.
- Using proton NMR integration rules for carbon-13 NMR.
- Applying the $n + 1$ rule without checking neighbouring equivalent protons.
- Treating one spectrum as conclusive when the structure needs several pieces of evidence.
