# 29 An Introduction to A Level Organic Chemistry - Review Checklist

## Source Scope

- [ ] I can state that this topic follows CAIE Chemistry 9701 Topic 29 and does not depend on a verified coursebook.
- [ ] I can explain how Topic 29 extends AS organic chemistry rather than replacing it.

## Functional Groups and Formulae

- [ ] I can recognise arene, halogenoarene, phenol, acyl chloride, amine, amide and amino acid functional groups.
- [ ] I can convert between general, structural, displayed and skeletal formulae in simple A Level examples.
- [ ] I remember that a displayed benzene ring is not expected where a benzene ring is part of the molecule.
- [ ] I can name simple aliphatic, cyclic and aromatic molecules within the syllabus scope.

## Mechanism Language

- [ ] I can describe electrophilic substitution as the characteristic mechanism of many arene reactions.
- [ ] I can describe addition-elimination as the mechanism used in acyl chloride reactions.
- [ ] I can avoid using alkene addition language for benzene reactions.

## Aromatic Bonding

- [ ] I can describe benzene as planar with $sp^2$ carbon atoms.
- [ ] I can distinguish $\sigma$ bonds from the delocalised $\pi$ system.
- [ ] I can explain aromatic stabilisation and why substitution is favoured over addition.

## Optical Isomerism

- [ ] I can identify a chiral centre in simple molecules.
- [ ] I can define enantiomer, optically active and racemic mixture.
- [ ] I can explain the effect of each enantiomer on plane-polarised light.
- [ ] I can explain why chirality matters in drug synthesis.

## Ready to Move On

- [ ] I can use Topic 29 language when studying arenes, acyl chlorides, amines, amino acids and synthesis.
- [ ] I can connect each A Level functional group to at least one later topic where it is used.
