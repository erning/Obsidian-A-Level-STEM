# 36 Organic Synthesis - 中文讲义

## 来源范围

本讲义依据 CAIE Chemistry 9701 A Level Topic 36.1 Organic synthesis。这里不声称对齐任何教材。

## 与 AS 的连接

AS 的 organic synthesis 已经建立了较小的反应网络：alkanes、alkenes、halogenoalkanes、alcohols、carbonyl compounds、carboxylic acids、esters、nitriles 和 amines。A Level 保留这些反应，并加入 arenes、phenols、acyl chlorides、amides、amino acids、diazonium salts、condensation polymers 和更高阶的 analysis。合成不是背孤立方程，而是在 syllabus 反应中选择可靠路线。

## 先识别官能团

遇到多官能团分子，先逐个识别官能团，再问：

- 给定试剂最可能与哪个官能团反应？
- 哪些官能团在该条件下可能保持不变？
- 反应改变碳骨架，还是只改变官能团？
- 是否有副产物或竞争反应？

例如 $\ce{C6H5COCl}$ 同时有 aromatic ring 和 acyl chloride。它与 ethanol 在室温反应时，主要是 acyl chloride 生成 ester，苯环不是反应中心。

## 常用反应地图

| 转化 | 试剂和条件 |
|---|---|
| benzene 到 nitrobenzene | concentrated $\ce{HNO3}$ 和 concentrated $\ce{H2SO4}$，$25$ 到 $60\,^\circ\mathrm{C}$ |
| nitrobenzene 到 phenylamine | hot $\ce{Sn}$ 和 concentrated $\ce{HCl}$，再加 $\ce{NaOH(aq)}$ |
| phenylamine 到 diazonium salt | $\ce{HNO2}$，或 $\ce{NaNO2}$ 加 dilute acid，低于 $10\,^\circ\mathrm{C}$ |
| diazonium salt 到 phenol | 与水温热 |
| benzene 到 halogenoarene | $\ce{Cl2}$ 或 $\ce{Br2}$，$\ce{AlCl3}$ 或 $\ce{AlBr3}$ |
| benzene 到 alkylbenzene | $\ce{CH3Cl}$、$\ce{AlCl3}$、heat |
| benzene 到 acylbenzene | $\ce{CH3COCl}$、$\ce{AlCl3}$、heat |
| alkylbenzene 到 benzoic acid | hot alkaline $\ce{KMnO4}$，再加 dilute acid |
| carboxylic acid 到 acyl chloride | $\ce{PCl3}$ 和 heat、$\ce{PCl5}$ 或 $\ce{SOCl2}$ |
| acyl chloride 到 ester | alcohol 或 phenol，室温 |
| acyl chloride 到 amide | ammonia 或 primary/secondary amine，室温 |
| halogenoalkane 到 amine | ethanolic $\ce{NH3}$，加热加压 |
| nitrile 到 amine | $\ce{LiAlH4}$ 或 $\ce{H2/Ni}$ |

这张表是路线图，不是机械背诵清单。

## 逆推路线

合成题常常从目标产物倒推更容易。

1. 找目标官能团。
2. 想它可由哪个直接前体生成。
3. 一步步倒推到题目给的起始物。
4. 检查每一步是否是 syllabus 反应。
5. 补全试剂、条件和副产物。

例如要制备 phenyl benzoate，最后一步可以是 benzoyl chloride 与 phenol 反应。Benzoyl chloride 可由 benzoic acid 制得；若有 methylbenzene，可先侧链氧化成 benzoic acid。

## 检查碳骨架

有些反应改变碳数，很多反应不改变。

- Halogenoalkane 与 cyanide 生成 nitrile 会增加一个碳。
- Methylbenzene 氧化成 benzoic acid 不增加碳，而是把侧链碳变成 $\ce{-COOH}$ 的碳。
- Friedel-Crafts acylation 会引入 acyl group。
- Acyl chloride 反应保留 acyl carbon，只是把 $\ce{-COCl}$ 中的氯换成亲核试剂部分。

路线合理前，必须检查碳数。

## 分析给定合成路线

每一步都要说明：

- reaction type，例如 electrophilic substitution、nucleophilic substitution、oxidation、reduction、condensation 或 addition-elimination；
- reagent 和 condition；
- organic product；
- 小分子副产物，例如 $\ce{HCl}$、$\ce{H2O}$ 或 $\ce{N2}$；
- 哪些官能团没有变化。

只写产物而不写条件，在合成题中通常不够。

## 多官能团选择性

多官能团分子需要判断选择性。Acyl chlorides 与水、alcohols、phenol、ammonia 和 amines 反应很快；phenol 和 phenylamine 的芳香环被强烈活化，取代条件比 benzene 温和；amino acids 的电荷状态取决于 pH。

当一个分子有多个反应位点时，要说明所选试剂为什么在该位置反应。

## 常见错误

- 只写产物，不写试剂和条件。
- 明明有 syllabus 路线，却使用范围外反应。
- 忘记 nitrile 路线增加一个碳。
- Acyl chloride 反应漏写 $\ce{HCl}$。
- 只从起始物正推，不会从目标产物逆推。
