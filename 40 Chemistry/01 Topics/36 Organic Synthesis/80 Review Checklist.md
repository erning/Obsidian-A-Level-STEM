# 36 Organic Synthesis - Review Checklist

## Source Scope

- [ ] I can state that this topic follows CAIE Chemistry 9701 Topic 36.1.
- [ ] I can explain how A Level synthesis builds on AS synthesis reactions.

## Functional-Group Analysis

- [ ] I can identify multiple functional groups in one molecule.
- [ ] I can predict which functional group reacts with a given reagent.
- [ ] I can spot when a carbon skeleton changes.
- [ ] I can identify likely by-products.

## Route Planning

- [ ] I can work backwards from a target molecule.
- [ ] I can build two-step and three-step routes using syllabus reactions.
- [ ] I can include reagents and conditions for every step.
- [ ] I can avoid using reactions outside the syllabus unless explicitly justified.

## Reaction Types

- [ ] I can name electrophilic substitution, nucleophilic substitution, oxidation, reduction, condensation and addition-elimination steps.
- [ ] I can analyse a given route in terms of reaction type, reagents and possible by-products.

## Ready to Combine Topics

- [ ] I can connect arenes, phenols, acyl chlorides, amines, amides and amino acids in routes.
- [ ] I can use analytical evidence to check whether a synthesis likely produced the intended compound.
