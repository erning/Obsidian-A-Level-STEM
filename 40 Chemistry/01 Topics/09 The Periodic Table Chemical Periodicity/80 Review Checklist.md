# The Periodic Table: Chemical Periodicity - Review Checklist

## Recall

- [ ] I can write the Period 3 sequence $\ce{Na, Mg, Al, Si, P, S, Cl, Ar}$.
- [ ] I can describe and explain the decrease in atomic radius across Period 3.
- [ ] I can explain cation and anion ionic radius trends separately.
- [ ] I know which Period 3 elements are metallic, giant covalent, simple molecular and monatomic.
- [ ] I can describe the melting point trend using structure and bonding.
- [ ] I can describe the electrical conductivity trend using mobile charged particles.

## Equations

- [ ] I can write equations for the specified reactions with oxygen to form $\ce{Na2O}$, $\ce{MgO}$, $\ce{Al2O3}$, $\ce{P4O10}$ and $\ce{SO2}$.
- [ ] I can write equations for the specified reactions with chlorine to form $\ce{NaCl}$, $\ce{MgCl2}$, $\ce{AlCl3}$, $\ce{SiCl4}$ and $\ce{PCl5}$.
- [ ] I can write equations for sodium with cold water, magnesium with cold water and magnesium with steam.
- [ ] I can write equations for $\ce{Na2O}$, $\ce{MgO}$, $\ce{P4O10}$, $\ce{SO2}$ and $\ce{SO3}$ with water where reactions occur.
- [ ] I can write equations showing amphoteric behaviour of $\ce{Al2O3}$ and $\ce{Al(OH)3}$.
- [ ] I can write hydrolysis equations for $\ce{SiCl4}$ and $\ce{PCl5}$.

## Explanations

- [ ] I can link oxide acid-base behaviour to metallic or non-metallic character.
- [ ] I can link chloride hydrolysis to covalent character and electronegativity.
- [ ] I can assign oxidation numbers in the named oxides and chlorides.
- [ ] I can explain why $\ce{Al2O3}$ and $\ce{Al(OH)3}$ are amphoteric.
- [ ] I can predict likely pH values after named oxides or chlorides react with water.

## Common Errors to Avoid

- [ ] I do not describe ionic radius as one smooth trend across the whole period.
- [ ] I do not say silicon melts by overcoming intermolecular forces.
- [ ] I do not treat all Period 3 chlorides as neutral salts.
- [ ] I do not confuse acidic oxides with acids already dissolved in water.
- [ ] I do not forget state symbols when an observation or practical context needs them.

## Ready to Move On

- [ ] I can solve a data question that asks me to identify an unknown Period 3 element.
- [ ] I can compare Period 3 trends with Group 2 and Group 17 trends.
- [ ] I can use balanced equations to support written explanations, rather than giving descriptions alone.
