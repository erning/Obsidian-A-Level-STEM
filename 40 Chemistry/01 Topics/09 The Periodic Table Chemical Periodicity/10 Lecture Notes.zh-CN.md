# The Periodic Table: Chemical Periodicity - Lecture Notes zh-CN

## 学习范围

本节对应 CAIE Chemistry 9701 AS Level 第 9 节：The Periodic Table: chemical periodicity。重点是第 3 周期元素的物理性质周期性、指定反应、氧化物和氯化物的酸碱行为，以及用周期性推断未知元素性质。这里不声称与任何 Chemistry coursebook 对齐。

第 3 周期顺序是：

$$
\ce{Na, Mg, Al, Si, P, S, Cl, Ar}
$$

## 什么是周期性

周期性指元素性质随电子结构重复而出现的规律。沿第 3 周期从左到右，每个元素依次多 1 个质子，也多 1 个电子。新增电子进入同一主能级，屏蔽作用增加不大，但核电荷逐渐增加，所以原子核对外层电子的吸引增强。

学习时不要只背趋势，要把性质和原因连起来：

| 性质 | 解释时要抓住的原因 |
|---|---|
| 原子半径 | 核电荷、屏蔽作用、原子核对外层电子的吸引 |
| 离子半径 | 失电子或得电子、电子层数、等电子离子中的核电荷 |
| 熔点 | 结构和成键 |
| 导电性 | 是否有可移动的带电粒子 |
| 氧化物和氯化物性质 | 成键、电负性、酸碱性 |

## 原子半径

第 3 周期从钠到氯，原子半径总体减小。原因是质子数增加，而新增电子进入同一外层，屏蔽作用没有大幅增加。有效核电荷增强，外层电子被拉得更靠近原子核。

氩也符合原子大小变小的大趋势，不过稀有气体半径的测量方式与共价半径不同。答题时重点说明“核电荷增加，而屏蔽作用变化较小”。

## 离子半径

离子半径不能简单说成“从左到右一直减小”，因为常见离子的电荷和电子数不同。

阳离子：

$$
\ce{Na+ > Mg^{2+} > Al^{3+}}
$$

这 3 个离子都是 10 个电子的等电子离子。质子数从 $\ce{Na+}$ 到 $\ce{Al^{3+}}$ 增加，同一团电子受到更强吸引，所以半径减小。

常见阴离子：

$$
\ce{P^{3-} > S^{2-} > Cl-}
$$

这些离子都是 18 个电子的等电子离子。质子数增加，半径减小。

阳离子比对应原子小，因为失去外层电子后电子层减少。阴离子比对应原子大，因为得电子后电子间排斥增强。

## 第 3 周期熔点

熔点主要由结构和成键决定：

| 元素 | 结构 | 熔点解释 |
|---|---|---|
| $\ce{Na, Mg, Al}$ | 巨型金属结构 | 从 $\ce{Na}$ 到 $\ce{Al}$，阳离子电荷增加，离域电子更多，金属键增强 |
| $\ce{Si}$ | 巨型共价结构 | 需要破坏大量强共价键，所以熔点很高 |
| $\ce{P, S, Cl, Ar}$ | 简单分子或单原子结构 | 熔化时只需克服弱的 instantaneous dipole-induced dipole forces |

硅的熔点最高，因为它是巨型共价结构。硫通常比磷和氯熔点高，因为 $\ce{S8}$ 分子比 $\ce{P4}$ 和 $\ce{Cl2}$ 更大、电子更多，分子间瞬时偶极-诱导偶极作用更强。

## 导电性

$\ce{Na}$、$\ce{Mg}$、$\ce{Al}$ 能导电，因为金属键中有可移动的离域电子。从钠到铝，每个原子提供的离域电子更多，金属阳离子与电子海的吸引也更强，所以导电性总体增强。

$\ce{Si}$ 是半导体。它的巨型共价结构在室温下没有大量可移动带电粒子，但在特定条件下导电性可以提高。

$\ce{P}$、$\ce{S}$、$\ce{Cl}$ 和 $\ce{Ar}$ 不导电，因为没有可移动离子或离域电子。

## 与氧气反应

大纲指定的产物如下：

| 元素 | 产物 | 方程式 |
|---|---|---|
| Sodium | Sodium oxide | $\ce{4Na(s) + O2(g) -> 2Na2O(s)}$ |
| Magnesium | Magnesium oxide | $\ce{2Mg(s) + O2(g) -> 2MgO(s)}$ |
| Aluminium | Aluminium oxide | $\ce{4Al(s) + 3O2(g) -> 2Al2O3(s)}$ |
| Phosphorus | Phosphorus(V) oxide | $\ce{4P(s) + 5O2(g) -> P4O10(s)}$ |
| Sulfur | Sulfur dioxide | $\ce{S(s) + O2(g) -> SO2(g)}$ |

左侧金属氧化物主要是离子化合物，右侧非金属氧化物主要是共价化合物。$\ce{Al2O3}$ 要特别注意，因为它是两性氧化物。

## 与氯气反应

大纲指定的产物如下：

| 元素 | 产物 | 方程式 |
|---|---|---|
| Sodium | Sodium chloride | $\ce{2Na(s) + Cl2(g) -> 2NaCl(s)}$ |
| Magnesium | Magnesium chloride | $\ce{Mg(s) + Cl2(g) -> MgCl2(s)}$ |
| Aluminium | Aluminium chloride | $\ce{2Al(s) + 3Cl2(g) -> 2AlCl3(s)}$ |
| Silicon | Silicon tetrachloride | $\ce{Si(s) + 2Cl2(g) -> SiCl4(l)}$ |
| Phosphorus | Phosphorus(V) chloride | $\ce{2P(s) + 5Cl2(g) -> 2PCl5(s)}$ |

沿周期从左到右，氯化物从以离子性为主逐渐变为以共价性为主。

## 钠、镁与水反应

钠与冷水剧烈反应：

$$
\ce{2Na(s) + 2H2O(l) -> 2NaOH(aq) + H2(g)}
$$

生成 $\ce{NaOH}$，所以溶液强碱性。

镁与冷水反应很慢，因为表面会形成难溶的 $\ce{Mg(OH)2}$：

$$
\ce{Mg(s) + 2H2O(l) -> Mg(OH)2(s) + H2(g)}
$$

镁与水蒸气反应更明显：

$$
\ce{Mg(s) + H2O(g) -> MgO(s) + H2(g)}
$$

## 氧化数

氧化数可以从化合物中氧或氯的常见氧化数推出。氧通常为 -2，氯通常为 -1，中性化合物总氧化数为 0。

| 化合物 | 第 3 周期元素的氧化数 |
|---|---:|
| $\ce{Na2O}$ | +1 |
| $\ce{MgO}$ | +2 |
| $\ce{Al2O3}$ | +3 |
| $\ce{P4O10}$ | +5 |
| $\ce{SO2}$ | +4 |
| $\ce{SO3}$ | +6 |
| $\ce{NaCl}$ | +1 |
| $\ce{MgCl2}$ | +2 |
| $\ce{AlCl3}$ | +3 |
| $\ce{SiCl4}$ | +4 |
| $\ce{PCl5}$ | +5 |

## 氧化物与水

| 氧化物 | 与水反应 | 溶液大致 pH |
|---|---|---|
| $\ce{Na2O}$ | $\ce{Na2O(s) + H2O(l) -> 2NaOH(aq)}$ | 强碱性 |
| $\ce{MgO}$ | $\ce{MgO(s) + H2O(l) -> Mg(OH)2(s)}$ | 弱碱性 |
| $\ce{Al2O3}$ | 与水无简单反应 | 近中性 |
| $\ce{SiO2}$ | 与水不反应 | 近中性 |
| $\ce{P4O10}$ | $\ce{P4O10(s) + 6H2O(l) -> 4H3PO4(aq)}$ | 酸性 |
| $\ce{SO2}$ | $\ce{SO2(g) + H2O(l) <=> H2SO3(aq)}$ | 酸性 |
| $\ce{SO3}$ | $\ce{SO3(g) + H2O(l) -> H2SO4(aq)}$ | 强酸性 |

趋势是：左侧碱性氧化物，中间出现两性或不与水反应的氧化物，右侧酸性氧化物。这与元素从金属性到非金属性、化合物从离子性到共价性的变化有关。

## 氧化物和氢氧化物的酸碱行为

碱性氧化物与酸反应：

$$
\ce{Na2O(s) + 2HCl(aq) -> 2NaCl(aq) + H2O(l)}
$$

$$
\ce{MgO(s) + 2HCl(aq) -> MgCl2(aq) + H2O(l)}
$$

酸性氧化物与碱反应：

$$
\ce{SO2(g) + 2NaOH(aq) -> Na2SO3(aq) + H2O(l)}
$$

$$
\ce{SO3(g) + 2NaOH(aq) -> Na2SO4(aq) + H2O(l)}
$$

$$
\ce{P4O10(s) + 12NaOH(aq) -> 4Na3PO4(aq) + 6H2O(l)}
$$

$\ce{Al2O3}$ 和 $\ce{Al(OH)3}$ 是两性的，既能与酸反应，也能与碱反应：

$$
\ce{Al2O3(s) + 6HCl(aq) -> 2AlCl3(aq) + 3H2O(l)}
$$

$$
\ce{Al2O3(s) + 2NaOH(aq) + 3H2O(l) -> 2NaAl(OH)4(aq)}
$$

$$
\ce{Al(OH)3(s) + 3HCl(aq) -> AlCl3(aq) + 3H2O(l)}
$$

$$
\ce{Al(OH)3(s) + NaOH(aq) -> NaAl(OH)4(aq)}
$$

氢氧化物也体现趋势：$\ce{NaOH}$ 是可溶强碱，$\ce{Mg(OH)2}$ 是难溶碱，$\ce{Al(OH)3}$ 是两性氢氧化物。

## 氯化物与水

指定氯化物从基本中性的离子化合物，逐渐过渡到会强烈水解并产生酸性溶液的共价氯化物。

| 氯化物 | 与水作用 | 溶液大致 pH |
|---|---|---|
| $\ce{NaCl}$ | 溶解但几乎不水解 | 约 7 |
| $\ce{MgCl2}$ | 溶解后溶液略呈酸性 | 略低于 7 |
| $\ce{AlCl3}$ | 水解，溶液酸性 | 酸性 |
| $\ce{SiCl4}$ | 剧烈水解，产生 $\ce{HCl}$ 烟雾 | 强酸性 |
| $\ce{PCl5}$ | 水解生成磷酸和 $\ce{HCl}$ | 强酸性 |

常用方程式：

$$
\ce{SiCl4(l) + 2H2O(l) -> SiO2(s) + 4HCl(aq)}
$$

$$
\ce{PCl5(s) + 4H2O(l) -> H3PO4(aq) + 5HCl(aq)}
$$

可以用下面的简化式表示 $\ce{AlCl3}$ 水解：

$$
\ce{AlCl3(s) + 3H2O(l) -> Al(OH)3(s) + 3HCl(aq)}
$$

氯化物越偏共价，越容易发生水解并产生酸性溶液。

## 用周期性推断未知元素

遇到未知元素题，可以按这些问题判断：

1. 物质是金属结构、巨型共价结构、简单分子，还是单原子结构？
2. 固体能否导电？
3. 氧化物是碱性、两性还是酸性？
4. 氯化物是否与水反应并使溶液呈酸性？
5. 氧化数是否符合族序数或外层电子数？

例如，某元素的氧化物为两性，氯化物溶于水后呈酸性，在第 3 周期中很可能指向 aluminium。

## 快速自测

- 能否解释为什么第 3 周期原子半径减小？
- 能否用结构和成键解释硅的熔点为什么高于铝？
- 能否写出指定元素与氧气、氯气反应的方程式？
- 能否判断 $\ce{Na2O}$、$\ce{MgO}$、$\ce{Al2O3}$、$\ce{P4O10}$、$\ce{SO2}$、$\ce{SO3}$ 的酸碱性？
- 能否预测 $\ce{SiCl4}$ 或 $\ce{PCl5}$ 加入水后的 pH？
