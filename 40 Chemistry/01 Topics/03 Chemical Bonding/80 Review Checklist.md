# Chemical Bonding - Review Checklist

## Recall

- [ ] I can define electronegativity and explain its trends across a period and down a group.
- [ ] I can define ionic bonding, metallic bonding, covalent bonding and coordinate bonding.
- [ ] I can state the difference between $\sigma$ and $\pi$ bonds.
- [ ] I can define bond energy and bond length.
- [ ] I can state the required VSEPR shapes and bond angles.
- [ ] I can describe instantaneous dipole-induced dipole, permanent dipole-permanent dipole and hydrogen bonding.
- [ ] I know that hydrogen bonding is limited here to $\ce{N-H}$ and $\ce{O-H}$ groups.

## Skills

- [ ] I can use electronegativity values to predict ionic, covalent and polar covalent bonding.
- [ ] I can describe bonding in $\ce{NaCl}$, $\ce{MgO}$ and $\ce{CaF2}$.
- [ ] I can draw or describe dot-and-cross diagrams for ionic, covalent and coordinate species.
- [ ] I can explain coordinate bonding in $\ce{NH4+}$ and $\ce{Al2Cl6}$.
- [ ] I can count $\sigma$ and $\pi$ bonds in specified molecules.
- [ ] I can use hybridisation to describe $sp$, $sp^2$ and $sp^3$ orbitals.
- [ ] I can predict shapes and bond angles for molecules and analogous ions using VSEPR theory.
- [ ] I can decide whether a molecule is polar by combining bond polarity with shape.
- [ ] I can explain melting point, boiling point, surface tension and density using intermolecular forces.

## Common Errors

- [ ] I do not call intermolecular forces "covalent bonds".
- [ ] I do not assume a molecule is polar just because it has polar bonds.
- [ ] I count multiple bonds as one electron region for VSEPR shape.
- [ ] I include lone pairs when predicting shape and bond angle.
- [ ] I remember that a coordinate bond is still a covalent bond after it has formed.
- [ ] I do not say covalent bonds inside simple molecules break during boiling.

## Self-Checks

- [ ] Given a pair of atoms, I can predict the likely bond type from electronegativity.
- [ ] Given a central atom, I can count bonding pairs and lone pairs before naming the shape.
- [ ] Given a molecule, I can identify the strongest intermolecular force present.
- [ ] Given two simple molecular substances, I can compare boiling points using intermolecular forces.
- [ ] Given a double or triple bond, I can identify the $\sigma$ and $\pi$ components.

## Connections

- [ ] I can connect electronic configuration to ion formation and covalent bonding.
- [ ] I can connect bonding type to the structures studied in States of Matter.
- [ ] I can connect $\sigma$ and $\pi$ bonding to organic molecules such as alkenes and nitriles.

## Ready When

- [ ] I can explain a physical property by naming the particles, the structure and the attraction involved.
- [ ] I can draw dot-and-cross diagrams with charges, lone pairs and coordinate bonds where needed.
- [ ] I can predict VSEPR shape without memorising only examples.
- [ ] I can separate bonds within particles from forces between particles in every explanation.
