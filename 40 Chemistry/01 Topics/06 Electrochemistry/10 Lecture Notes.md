# Electrochemistry - Lecture Notes

## Source Route

These notes follow CAIE Chemistry 9701 AS Level Physical chemistry, Topic 6, Electrochemistry: 6.1 Redox processes, electron transfer and changes in oxidation number (oxidation state). Electrolysis and standard electrode potentials are A Level Topic 24 in the verified syllabus, so this AS note treats them only as later applications of redox. No coursebook alignment is claimed.

## Redox As Electron Transfer

Redox reactions involve electron transfer. Oxidation is loss of electrons; reduction is gain of electrons.

The half-equation:

$$
\ce{Mg -> Mg^{2+} + 2e-}
$$

shows oxidation because magnesium loses electrons.

The half-equation:

$$
\ce{Cu^{2+} + 2e- -> Cu}
$$

shows reduction because copper(II) ions gain electrons.

When these half-equations are combined, electrons lost by one species are gained by another:

$$
\ce{Mg + Cu^{2+} -> Mg^{2+} + Cu}
$$

Electron transfer is especially clear in ionic equations and half-equations. In molecular equations, oxidation number changes are often a better guide.

## Oxidation Numbers

An oxidation number is a formal number assigned to an atom in a compound or ion. It is useful for tracking electron transfer even when bonding is covalent and no simple ions exist.

Use these rules first:

| Rule | Example |
|---|---|
| An uncombined element has oxidation number 0 | $\ce{Mg}$, $\ce{O2}$ and $\ce{Cl2}$ all contain atoms with oxidation number 0 |
| A monatomic ion has oxidation number equal to its charge | $\ce{Na+}$ is +1; $\ce{S^{2-}}$ is -2 |
| The sum of oxidation numbers in a neutral compound is 0 | $\ce{H2O}$ totals 0 |
| The sum of oxidation numbers in an ion equals the charge on the ion | $\ce{SO4^{2-}}$ totals -2 |
| Group 1 metals are usually +1 | $\ce{Na}$ in $\ce{NaCl}$ is +1 |
| Group 2 metals are usually +2 | $\ce{Mg}$ in $\ce{MgO}$ is +2 |
| Fluorine is -1 in its compounds | $\ce{F}$ in $\ce{NaF}$ is -1 |
| Hydrogen is usually +1 with non-metals | $\ce{H}$ in $\ce{HCl}$ is +1 |
| Oxygen is usually -2 | $\ce{O}$ in $\ce{CO2}$ is -2 |

There are exceptions, such as oxygen in peroxides, but the common AS calculations normally signal any unusual case clearly.

For example, in $\ce{SO4^{2-}}$, oxygen is usually -2. Four oxygen atoms contribute $4(-2) = -8$. The total charge is -2, so sulfur must be +6:

$$
x + 4(-2) = -2
$$

$$
x = +6
$$

## Oxidation, Reduction And Redox By Oxidation Number

Oxidation is an increase in oxidation number. Reduction is a decrease in oxidation number.

Consider:

$$
\ce{Zn + Cu^{2+} -> Zn^{2+} + Cu}
$$

Zinc changes from 0 to +2, so zinc is oxidised. Copper changes from +2 to 0, so copper is reduced. Because oxidation and reduction occur together, this is a redox reaction.

A reaction is not redox if no oxidation numbers change. For example, acid-base neutralisation usually involves ion combination, not electron transfer:

$$
\ce{H+(aq) + OH-(aq) -> H2O(l)}
$$

Hydrogen remains +1 and oxygen remains -2, so this ionic equation is not redox.

## Oxidising Agents And Reducing Agents

An oxidising agent causes another species to be oxidised. It accepts electrons and is reduced.

A reducing agent causes another species to be reduced. It donates electrons and is oxidised.

In:

$$
\ce{Zn + Cu^{2+} -> Zn^{2+} + Cu}
$$

$\ce{Cu^{2+}}$ is the oxidising agent because it oxidises zinc and is itself reduced. $\ce{Zn}$ is the reducing agent because it reduces $\ce{Cu^{2+}}$ and is itself oxidised.

This is a common source of reversed answers. Always identify what happens to the agent itself:

- oxidising agent is reduced;
- reducing agent is oxidised.

## Disproportionation

Disproportionation is a redox reaction in which the same element is both oxidised and reduced.

For example:

$$
\ce{Cl2 + 2OH- -> Cl- + ClO- + H2O}
$$

Chlorine starts at oxidation number 0 in $\ce{Cl2}$. In $\ce{Cl-}$, chlorine is -1, so it has been reduced. In $\ce{ClO-}$, oxygen is -2 and the ion has charge -1, so chlorine is +1, meaning it has been oxidised. The same starting element has gone both down and up in oxidation number, so the reaction is disproportionation.

## Balancing Equations With Oxidation Number Changes

Oxidation number changes help determine coefficients. The total increase in oxidation number must equal the total decrease.

For example, in acidified solution, $\ce{Fe^{2+}}$ is oxidised to $\ce{Fe^{3+}}$ and $\ce{MnO4-}$ is reduced to $\ce{Mn^{2+}}$.

Iron:

$$
\ce{Fe^{2+} -> Fe^{3+} + e-}
$$

Manganese changes from +7 in $\ce{MnO4-}$ to +2 in $\ce{Mn^{2+}}$, so each manganese gains 5 electrons:

$$
\ce{MnO4- + 8H+ + 5e- -> Mn^{2+} + 4H2O}
$$

Five iron(II) ions are needed to supply five electrons:

$$
\ce{5Fe^{2+} + MnO4- + 8H+ -> 5Fe^{3+} + Mn^{2+} + 4H2O}
$$

The final equation is balanced for atoms and charge. Charge on the left is $5(+2) - 1 + 8(+1) = +17$. Charge on the right is $5(+3) + 2 = +17$.

## Roman Numerals In Names

Roman numerals can show the magnitude of an oxidation number where an element has more than one common oxidation state. The numeral is written in brackets after the element name.

Examples:

- iron(II) ions are $\ce{Fe^{2+}}$;
- iron(III) ions are $\ce{Fe^{3+}}$;
- copper(I) oxide is $\ce{Cu2O}$ because each copper is +1;
- copper(II) oxide is $\ce{CuO}$ because copper is +2;
- manganate(VII) contains manganese in oxidation state +7, as in $\ce{MnO4-}$.

Roman numerals do not show the number of atoms. They show oxidation number.

## Working Routine

For an unfamiliar redox equation:

1. Assign oxidation numbers to the elements that may change.
2. Identify increases and decreases.
3. Name the species oxidised and reduced.
4. Name the oxidising agent and reducing agent.
5. If balancing, make total oxidation number increase equal total decrease.
6. Check atoms and total charge.

## Common Mistakes

Do not assume oxygen gain or hydrogen loss is enough evidence. CAIE questions often expect oxidation number or electron-transfer language.

Do not confuse the charge on a polyatomic ion with the oxidation number of each atom inside it. In $\ce{NO3-}$, nitrogen is not -1; the whole ion is -1.

Do not call a species an oxidising agent just because it is oxidised. The oxidising agent is reduced.

Do not forget disproportionation. If one element in a reactant appears in two products with different oxidation numbers, check whether both oxidation and reduction have occurred.
