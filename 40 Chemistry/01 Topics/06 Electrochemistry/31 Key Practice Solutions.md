# Electrochemistry - Key Practice Solutions

1. Oxidation is loss of electrons. Reduction is gain of electrons.

2. Oxidation is an increase in oxidation number. Reduction is a decrease in oxidation number.

3. For $\ce{SO2}$:

$$
x + 2(-2) = 0
$$

$$
x = +4
$$

Sulfur is +4.

For $\ce{SO4^{2-}}$:

$$
x + 4(-2) = -2
$$

$$
x = +6
$$

Sulfur is +6.

For $\ce{MnO4-}$:

$$
x + 4(-2) = -1
$$

$$
x = +7
$$

Manganese is +7.

For $\ce{NH4+}$:

$$
x + 4(+1) = +1
$$

$$
x = -3
$$

Nitrogen is -3.

4. Magnesium changes from 0 to +2, so magnesium is oxidised. Hydrogen changes from +1 in $\ce{H+}$ to 0 in $\ce{H2}$, so hydrogen is reduced. $\ce{H+}$ is the oxidising agent because it is reduced. $\ce{Mg}$ is the reducing agent because it is oxidised.

5. In:

$$
\ce{Ag+(aq) + Cl-(aq) -> AgCl(s)}
$$

silver remains +1 and chlorine remains -1. No oxidation numbers change, so there is no oxidation or reduction.

6. Oxygen in $\ce{H2O2}$ is -1 because hydrogen is +1. In $\ce{H2O}$, oxygen is -2, so oxygen has been reduced. In $\ce{O2}$, oxygen is 0, so oxygen has been oxidised. The same element has been both oxidised and reduced, so the reaction is disproportionation.

7. Iodide ions are oxidised:

$$
\ce{2I- -> I2 + 2e-}
$$

Chlorine is reduced:

$$
\ce{Cl2 + 2e- -> 2Cl-}
$$

Overall:

$$
\ce{2I- + Cl2 -> I2 + 2Cl-}
$$

8. Iron changes from +2 to +3, losing one electron per ion. Chromium changes from +6 in $\ce{Cr2O7^{2-}}$ to +3 in $\ce{Cr^{3+}}$. Two chromium atoms each gain 3 electrons, so 6 electrons are gained in total.

The balanced equation is:

$$
\ce{Cr2O7^{2-} + 6Fe^{2+} + 14H+ -> 2Cr^{3+} + 6Fe^{3+} + 7H2O}
$$

Charge check: left side is $-2 + 12 + 14 = +24$; right side is $6 + 18 = +24$.

9. Copper(II) means $\ce{Cu^{2+}}$, and chloride is $\ce{Cl-}$, so copper(II) chloride is $\ce{CuCl2}$. In $\ce{FeO}$, oxygen is -2, so iron is +2. The name is iron(II) oxide.

10. Electrolysis and electrode potential questions use oxidation and reduction half-equations. AS redox skills show which species loses or gains electrons, how to balance electrons, and how to connect electron transfer with oxidation number changes.
