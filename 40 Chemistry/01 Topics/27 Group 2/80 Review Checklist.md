# 27 Group 2 - Review Checklist

## Core Knowledge

- [ ] I can state that this A Level topic covers Group 2 metals from $\ce{Mg}$ to $\ce{Ba}$ and their compounds.
- [ ] I can state that $\ce{M^{2+}}$ ionic radius increases down Group 2.
- [ ] I can link increasing ionic radius to decreasing charge density.
- [ ] I can define polarisation in this context as distortion of a large anion's electron cloud by a cation.

## Thermal Stability

- [ ] I can write $\ce{MCO3(s) -> MO(s) + CO2(g)}$.
- [ ] I can write $\ce{2M(NO3)2(s) -> 2MO(s) + 4NO2(g) + O2(g)}$.
- [ ] I can state that Group 2 carbonates become more thermally stable from $\ce{MgCO3}$ to $\ce{BaCO3}$.
- [ ] I can state that Group 2 nitrates become more thermally stable from $\ce{Mg(NO3)2}$ to $\ce{Ba(NO3)2}$.
- [ ] I can explain both trends using ionic radius, charge density, polarisation, and anion stability.

## Solubility and Energy

- [ ] I can describe dissolution as a balance between lattice energy and enthalpy change of hydration.
- [ ] I can state that Group 2 hydroxides become more soluble down the group.
- [ ] I can explain hydroxide solubility by saying that the decrease in lattice energy is more important than the decrease in hydration enthalpy.
- [ ] I can state that Group 2 sulfates become less soluble down the group.
- [ ] I can explain sulfate solubility by saying that the decrease in hydration enthalpy is more important than the decrease in lattice energy.
- [ ] I can use $\Delta H^\circ_\mathrm{sol}$ correctly in qualitative explanations.

## Exam Readiness

- [ ] I separate observation, trend, and explanation in written answers.
- [ ] I do not mix up the hydroxide and sulfate solubility trends.
- [ ] I do not explain thermal stability by radius alone.
- [ ] I can interpret data prompts about decomposition temperature or solubility.
- [ ] I can write concise answers using CAIE terms: ionic radius, polarisation, lattice energy, and enthalpy change of hydration.

## Ready to Connect

- [ ] I can connect Group 2 solubility to [Chemical Energetics](../23%20Chemical%20Energetics/00%20Overview.md).
- [ ] I can connect Group 2 trends to [The Periodic Table: Chemical Periodicity](../09%20The%20Periodic%20Table%20Chemical%20Periodicity/00%20Overview.md).
- [ ] I can revisit [AS Group 2](../10%20Group%202/00%20Overview.md) without confusing AS reaction descriptions with A Level energetic explanations.
