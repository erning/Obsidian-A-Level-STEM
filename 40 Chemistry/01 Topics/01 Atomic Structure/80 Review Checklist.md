# Atomic Structure - Review Checklist

## Recall

- [ ] I can state the relative charge, relative mass and position of protons, neutrons and electrons.
- [ ] I can define atomic number, proton number, mass number and nucleon number.
- [ ] I can define isotope using numbers of protons and neutrons.
- [ ] I can define shell, sub-shell, orbital, principal quantum number and ground state.
- [ ] I can state the number of orbitals and maximum electrons in $s$, $p$ and $d$ sub-shells.
- [ ] I can define first ionisation energy and write equations for successive ionisation energies.

## Skills

- [ ] I can calculate protons, neutrons and electrons in atoms and ions from isotope notation and charge.
- [ ] I can describe the behaviour of proton, neutron and electron beams in an electric field.
- [ ] I can explain atomic radius and ionic radius trends across a period and down a group.
- [ ] I can write full and shorthand electronic configurations for atoms and ions from hydrogen to krypton.
- [ ] I can use electrons-in-boxes notation for sub-shell filling.
- [ ] I can describe and sketch $s$ and $p$ orbital shapes.
- [ ] I can interpret successive ionisation energy data to deduce outer-shell electrons and group.

## Common Errors

- [ ] I do not confuse mass number with proton number.
- [ ] I adjust electron number correctly for ionic charge.
- [ ] I remove $4s$ electrons before $3d$ electrons when forming transition metal ions.
- [ ] I include gaseous state symbols in ionisation energy equations.
- [ ] I explain ionisation energy trends using nuclear charge, radius, shielding or spin-pair repulsion, not vague difficulty.
- [ ] I do not say isotopes have different chemical properties because their masses differ.

## Self-Checks

- [ ] Given $^{x}_{y}\ce{A^{q+}}$, I can explain what each symbol means.
- [ ] Given an isoelectronic series, I can identify the smallest ion using proton number.
- [ ] Given a Period 3 ionisation energy pattern, I can explain both the general increase and the dips.
- [ ] Given successive ionisation energies, I can locate the big jump and translate it into group position.

## Connections

- [ ] I can connect isotopes to relative atomic mass calculations in stoichiometry.
- [ ] I can connect electronic configurations to ionic, covalent and metallic bonding.
- [ ] I can connect atomic radius and ionisation energy to periodic trends.

## Ready When

- [ ] I can complete a particle-count table without trial and error.
- [ ] I can write configurations up to krypton confidently, including common ions.
- [ ] I can give a cause-and-effect explanation for radius and ionisation energy trends.
- [ ] I can use data, not memorised labels only, to infer structure from successive ionisation energies.
