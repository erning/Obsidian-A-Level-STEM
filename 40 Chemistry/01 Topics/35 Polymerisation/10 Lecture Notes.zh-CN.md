# 35 Polymerisation - Lecture Notes

## 范围

本讲义依据 CAIE Chemistry 9701 A Level Topic 35：condensation polymerisation、判断 polymerisation 类型，以及 degradable polymers。这里不声称对齐任何教材。

## 与 AS 的连接

AS 的 polymerisation 重点是 alkenes 的 addition polymerisation。双键打开，单体连成聚合物，不消除小分子。A Level 加入 condensation polymerisation：单体通常有两个官能团，连接时会消除小分子，例如 water 或 hydrogen chloride。判断类型时，先看官能团。

## Addition 还是 condensation

判断方法：

- Addition polymerisation 通常来自含 $\ce{C=C}$ 的 alkene monomers。重复单元含有单体的全部原子，不消除小分子。
- Condensation polymerisation 来自 difunctional monomers，例如 diols、dicarboxylic acids、dioyl chlorides、diamines、hydroxycarboxylic acids、aminocarboxylic acids 和 amino acids。每形成一个连接，都会消除一个小分子。

A Level 最常见的连接是 ester link 和 amide link。

## Polyesters

Polyesters 含 ester links，$\ce{-COO-}$。

Diol 与 dicarboxylic acid 反应：

$$
\ce{n HO-R-OH + n HOOC-R'-COOH -> [-O-R-OOC-R'-CO-]_{n} + 2n H2O}
$$

Diol 与 dioyl chloride 反应：

$$
\ce{n HO-R-OH + n ClOC-R'-COCl -> [-O-R-OOC-R'-CO-]_{n} + 2n HCl}
$$

Hydroxycarboxylic acid 可以自缩聚，因为同一个分子有 $\ce{-OH}$ 和 $\ce{-COOH}$：

$$
\ce{n HO-R-COOH -> [-O-R-CO-]_{n} + n H2O}
$$

画 repeat unit 时，要保留两端继续延伸的键。

## Polyamides

Polyamides 含 amide links，$\ce{-CONH-}$。Proteins 中的 peptide bonds 本质上也是 amide links。

Diamine 与 dicarboxylic acid 反应：

$$
\ce{n H2N-R-NH2 + n HOOC-R'-COOH -> [-NH-R-NHCO-R'-CO-]_{n} + 2n H2O}
$$

Diamine 与 dioyl chloride 反应：

$$
\ce{n H2N-R-NH2 + n ClOC-R'-COCl -> [-NH-R-NHCO-R'-CO-]_{n} + 2n HCl}
$$

Aminocarboxylic acids 或 amino acids 可以自缩聚：

$$
\ce{n H2N-R-COOH -> [-NH-R-CO-]_{n} + n H2O}
$$

## 从单体画 repeat unit

步骤：

1. 找出会反应的两个官能团。
2. 根据反应去掉 $\ce{H}$、$\ce{OH}$ 或 $\ce{Cl}$。
3. 形成 ester link 或 amide link。
4. 用括号圈出最小重复部分。
5. 在两端保留继续延伸的键。

## 从 polymer section 找 monomers

看到 ester link，$\ce{-COO-}$，就从这里切开；羰基一侧补回 $\ce{-COOH}$，氧一侧补回 $\ce{-OH}$。如果题目使用 dioyl chloride，也可把酸部分写成 acyl chloride 的形式。

看到 amide link，$\ce{-CONH-}$，也从这里切开；羰基一侧补回 $\ce{-COOH}$，氮一侧补回 $\ce{-NH2}$。

## Degradable polymers

Poly(alkenes) 的主链主要是强 $\ce{C-C}$ 键，化学性质很惰性，因此较难 biodegrade。

有些 polymers 可被 light 作用降解，因为光能打断键或产生更活泼的位置。

Polyesters 和 polyamides 可通过 acidic 或 alkaline hydrolysis 生物降解，因为 ester links 和 amide links 能被水解切断。判断降解性时，看主链中有没有可水解的官能团。

## 常见错误

- 以为只要有两个单体就是 addition polymerisation。
- 忘记 condensation polymerisation 会消除小分子。
- 画 repeat unit 时漏掉延伸键。
- 从错误位置切开聚合物，得到不可能的单体。
- 认为所有 polymers 都有类似的 biodegradability。
