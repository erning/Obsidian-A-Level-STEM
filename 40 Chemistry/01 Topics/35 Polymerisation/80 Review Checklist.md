# 35 Polymerisation - Review Checklist

## Source Scope

- [ ] I can state that this topic follows CAIE Chemistry 9701 Topic 35.
- [ ] I can explain how condensation polymerisation extends AS addition polymerisation.

## Condensation Polymerisation

- [ ] I can identify monomers that form polyesters.
- [ ] I can identify monomers that form polyamides.
- [ ] I can draw repeat units from a diol and dicarboxylic acid or dioyl chloride.
- [ ] I can draw repeat units from a diamine and dicarboxylic acid or dioyl chloride.
- [ ] I can handle hydroxycarboxylic acids, aminocarboxylic acids and amino acids.

## Reverse Reasoning

- [ ] I can identify monomers from a section of a condensation polymer.
- [ ] I can cut ester and amide links correctly.
- [ ] I can restore $\ce{-OH}$, $\ce{-COOH}$ and $\ce{-NH2}$ groups when deducing monomers.

## Polymerisation Type

- [ ] I can predict addition or condensation polymerisation from monomer structure.
- [ ] I can deduce polymerisation type from a polymer section.

## Degradability

- [ ] I can explain why poly(alkenes) are chemically inert and difficult to biodegrade.
- [ ] I can recognise that some polymers degrade by light.
- [ ] I can explain acidic and alkaline hydrolysis of polyesters and polyamides.

## Ready to Move On

- [ ] I can connect ester links to carboxylic acid derivative chemistry.
- [ ] I can connect amide and peptide links to nitrogen chemistry.
