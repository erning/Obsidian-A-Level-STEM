# Chemistry Topics Overview

This index follows the CAIE Chemistry 9701 syllabus topic order. Topics 1-22 are the AS Level route; A Level candidates study all topics, with topics 23-37 extending physical chemistry, inorganic chemistry, organic chemistry, synthesis, and analytical techniques.

The links below target each topic directory's `00 Overview.md`. Each topic overview links to the completed lecture notes, Chinese companion notes, worked examples, practice problems, solutions, and review checklist.
