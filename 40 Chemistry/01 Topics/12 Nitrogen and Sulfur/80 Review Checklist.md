# Nitrogen and Sulfur - Review Checklist

## Recall

- [ ] I can explain nitrogen's low reactivity using $\ce{N#N}$ triple bond strength.
- [ ] I can also mention that $\ce{N2}$ is non-polar.
- [ ] I can define a Bronsted-Lowry base as a proton acceptor.
- [ ] I can explain ammonia's basicity using its lone pair.
- [ ] I can describe $\ce{NH4+}$ as tetrahedral with four equivalent $\ce{N-H}$ bonds.
- [ ] I can state natural and man-made occurrences of $\ce{NO}$ and $\ce{NO2}$.
- [ ] I can name PAN as peroxyacetyl nitrate and a component of photochemical smog.

## Equations

- [ ] I can write $\ce{NH3 + H+ -> NH4+}$.
- [ ] I can write $\ce{NH3 + H2O <=> NH4+ + OH-}$.
- [ ] I can write $\ce{NH3 + HCl -> NH4Cl}$.
- [ ] I can write $\ce{NH4+ + OH- -> NH3 + H2O}$.
- [ ] I can write $\ce{N2 + O2 -> 2NO}$ and $\ce{2NO + O2 -> 2NO2}$.
- [ ] I can write catalytic converter equations for $\ce{NO}$ or $\ce{NO2}$ with $\ce{CO}$.
- [ ] I can write equations for nitric acid and sulfuric acid formation in acid rain.
- [ ] I can write equations showing $\ce{NO2}$ oxidising $\ce{SO2}$ and $\ce{NO}$ regenerating $\ce{NO2}$.

## Oxidation Numbers

- [ ] I know nitrogen is +2 in $\ce{NO}$ and +4 in $\ce{NO2}$.
- [ ] I know nitrogen is 0 in $\ce{N2}$.
- [ ] I know sulfur is +4 in $\ce{SO2}$ and +6 in $\ce{SO3}$ or $\ce{H2SO4}$.
- [ ] I can identify reduction of nitrogen oxides in catalytic converters.
- [ ] I can identify oxidation of $\ce{SO2}$ to $\ce{SO3}$.

## Common Errors to Avoid

- [ ] I do not say nitrogen is unreactive only because it is a gas.
- [ ] I do not confuse $\ce{NH3}$ with $\ce{NH4+}$.
- [ ] I do not forget that ammonia is a proton acceptor.
- [ ] I do not describe PAN formation without mentioning nitrogen oxides and unburned hydrocarbons.
- [ ] I do not omit sulfur dioxide when explaining nitrogen oxides' catalytic role in acid rain.

## Ready to Move On

- [ ] I can connect engine temperature, $\ce{NOx}$ formation, catalytic removal, photochemical smog and acid rain in one coherent explanation.
- [ ] I can support environmental explanations with balanced equations.
- [ ] I can use oxidation numbers to make the redox changes explicit.
