# Data and Periodic Table

This is a locator note for using chemical data. It is not a replacement for the official data section in the CAIE Chemistry 9701 syllabus or exam materials.

## Local Syllabus Data Section

The local 2025-2027 syllabus data section lists:

- important values, constants and standards;
- ionisation energies of selected elements;
- bond energies;
- standard electrode potentials at 298 K;
- Pauling electronegativity values;
- typical proton NMR chemical shift values relative to TMS;
- typical carbon-13 NMR chemical shift values relative to TMS;
- characteristic infrared absorption frequencies for selected bonds;
- the Periodic Table of Elements.

## Frequently Used Constants

Check the official data section before using these in assessed work.

| specific heat capacity of water | $c$ | $4.18\ \mathrm{kJ\ kg^{-1}\ K^{-1}}$ or $4.18\ \mathrm{J\ g^{-1}\ K^{-1}}$ |

## How to Use Data

- Write units before substituting into a calculation.
- Check whether a value applies at standard conditions, room conditions, or 298 K.
- Use bond energies as average values unless the question says otherwise.
- Use electrode potentials with attention to direction: reduction potentials are tabulated for reduction half-equations.
- In spectroscopy, match the data to all evidence given, not to one isolated peak.
