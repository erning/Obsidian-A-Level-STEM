# Planning, Analysis and Evaluation - Review Checklist

## Recall

- [ ] I can list the four Paper 5 skill areas.
- [ ] I can define independent, dependent and control variables.
- [ ] I can define reliability and validity and distinguish them.
- [ ] I can define a control experiment and a prediction.
- [ ] I can state the form $y = mx + c$ and what $m$ and $c$ mean.

## Skills

- [ ] I can write a prediction linking the independent and dependent variables.
- [ ] I can describe a method with apparatus, steps, measurements and a results table.
- [ ] I can choose quantities to plot that linearise a suspected relationship.
- [ ] I can find $m$ and $c$ from a straight-line graph.
- [ ] I can identify the main source of error and a matching improvement.
- [ ] I can write a conclusion that links data to chemistry.

## Common Errors

- [ ] I do not list apparatus without describing the steps.
- [ ] I do not state a prediction without naming the control variables.
- [ ] I do not plot the wrong quantities and lose the straight line.
- [ ] I do not suggest an improvement that does not target the main error.
- [ ] I do not say data "prove" a prediction rather than "support" it.
- [ ] I do not give a conclusion to more significant figures than the data justify.

## Self-Checks

- [ ] Given a relationship, I can pick axes that give a straight line.
- [ ] Given a method, I can name the largest source of error and an improvement.
- [ ] Given data, I can write a conclusion and a judgement of reliability.
- [ ] Given repeats, I can judge whether the range and replication are adequate.

## Connections

- [ ] I can link a plan to variables and safety in *Experimental Design and Safety*.
- [ ] I can link analysis to graphs and errors in *Measurement, Observation and Calculation*.
- [ ] I can link conclusions to observation-based reasoning in *Qualitative Analysis*.

## Ready When

- [ ] I can write a full Paper 5 plan end to end.
- [ ] I can turn raw data into a straight-line graph and read $m$ and $c$.
- [ ] I can evaluate a method: anomalies, main error, realistic improvement, validity.
