# Qualitative Analysis - Lecture Notes

## 范围

本讲对应 CAIE Chemistry 9701 Syllabus 2025-2027 第 5 节 *Practical assessment*：第 64 页的 qualitative analysis 方法、同页的 organic analysis tests，以及第 69-70 页的 *Qualitative analysis notes*（阳离子、阴离子、气体和元素）。Paper 3 中有一道观察题，考试时会随卷提供 Qualitative analysis notes。

技能是先规范地观察，再谨慎地推断：先看现象，再下结论。

## 良好的定性分析习惯

- 所有未知物都要小心对待。
- 取 **合适量** 的待测物；既不浪费也不过量。
- 只加 **指定量** 试剂，若需过量再继续加。
- 注意安全：加热时用试管夹，固体在硬质试管中加热。
- **全部** 现象都要记下来，包括“no change”或“remains a colourless solution”。
- 产生沉淀时要用 **过量** $\ce{NaOH(aq)}$ 或 $\ce{NH3(aq)}$ 检验它在过量中的溶解性。
- 凡是有 effervescence（冒气泡）都要鉴定 **气体**。

阴性观察也是证据。“加氨水无沉淀”能排除若干阳离子，必须记下。

## 记录与下结论

把观察和结论分开。

| 观察 | 它是什么 | 结论（举例） |
| --- | --- | --- |
| 白色沉淀，溶于过量 $\ce{NaOH(aq)}$ | 一个被记录的现象 | 可能是 $\ce{Al^{3+}}$ 或 $\ce{Zn^{2+}}$ |
| 气体使湿润红石蕊变蓝 | 一个检验结果 | 氨气 |
| 加稀酸无变化 | 一个阴性结果 | 碳酸盐可能性低 |

按检验顺序记录现象。只有当现象支持时才下结论；证据尚不能区分时，保留多种可能性。

## 用 NaOH(aq) 和 NH3(aq) 检验阳离子

加氢氧化钠溶液和氨水（含过量）是鉴定阳离子的主要方法。在过量中的表现能区分初始沉淀相同的离子。

| 阳离子 | 与 $\ce{NaOH(aq)}$ | 与 $\ce{NH3(aq)}$ |
| --- | --- | --- |
| $\ce{Al^{3+}}$ | 白色沉淀，溶于过量 | 白色沉淀，不溶于过量 |
| $\ce{Ca^{2+}}$ | 白色沉淀（很稀时可能无） | 无沉淀 |
| $\ce{Mg^{2+}}$ | 白色沉淀，不溶于过量 | 白色沉淀，不溶于过量 |
| $\ce{Zn^{2+}}$ | 白色沉淀，溶于过量 | 白色沉淀，溶于过量 |
| $\ce{Fe^{2+}}$ | 绿色沉淀，空气中变棕 | 绿色沉淀，空气中变棕 |
| $\ce{Fe^{3+}}$ | 红棕色沉淀，不溶于过量 | 红棕色沉淀，不溶于过量 |
| $\ce{Cu^{2+}}$ | 浅蓝色沉淀，不溶于过量 | 浅蓝色沉淀，溶于过量得到深蓝色溶液 |
| $\ce{Cr^{3+}}$ | 灰绿色沉淀，溶于过量得到深绿色溶液 | 灰绿色沉淀，不溶于过量 |
| $\ce{NH4+}$ | 无沉淀；加热放出氨气 | n/a |

过量中的溶解性常常是决定性线索。$\ce{Al^{3+}}$ 和 $\ce{Zn^{2+}}$ 都给出溶于过量 $\ce{NaOH(aq)}$ 的白色沉淀，但 $\ce{Al^{3+}}$ 在过量 $\ce{NH3(aq)}$ 中不溶，$\ce{Zn^{2+}}$ 溶解。$\ce{Ca^{2+}}$ 与 $\ce{NH3(aq)}$ 不产生沉淀，可与 $\ce{Mg^{2+}}$ 区分。

铵离子不产生沉淀；与 $\ce{NaOH(aq)}$ 共热放出氨气，用湿润红石蕊变蓝检验。

## 阴离子检验

| 阴离子 | 检验与现象 |
| --- | --- |
| 碳酸根 $\ce{CO3^{2-}}$ | 与稀酸放出 $\ce{CO2}$ |
| 硫酸根 $\ce{SO4^{2-}}$ | 与 $\ce{Ba^{2+}(aq)}$ 生成白色沉淀，不溶于稀酸 |
| 亚硫酸根 $\ce{SO3^{2-}}$ | 与 $\ce{Ba^{2+}(aq)}$ 生成白色沉淀，溶于稀酸；使酸化 $\ce{KMnO4}$ 褪色 |
| 氯离子 $\ce{Cl-}$ | 与 $\ce{Ag+(aq)}$ 生成白色沉淀，溶于 $\ce{NH3(aq)}$ |
| 溴离子 $\ce{Br-}$ | 与 $\ce{Ag+(aq)}$ 生成奶油色/类白色沉淀，部分溶于 $\ce{NH3(aq)}$ |
| 碘离子 $\ce{I-}$ | 与 $\ce{Ag+(aq)}$ 生成浅黄色沉淀，不溶于 $\ce{NH3(aq)}$ |
| 硝酸根 $\ce{NO3-}$ | 与 $\ce{OH-}$ 和铝箔共热放出氨气 |
| 亚硝酸根 $\ce{NO2-}$ | 同硝酸根；也使酸化 $\ce{KMnO4}$ 褪色 |
| 硫代硫酸根 $\ce{S2O3^{2-}}$ | 与 $\ce{H+}$ 缓慢生成类白色/浅黄色沉淀 |

加硝酸银前先酸化，可排除干扰的碳酸根或氢氧根。区分硫酸根与亚硫酸根时，二者都生成钡的白色沉淀，但只有亚硫酸根的沉淀溶于稀酸，且只有亚硫酸根使酸化高锰酸钾褪色。

## 气体检验

| 气体 | 检验与现象 |
| --- | --- |
| 氨 $\ce{NH3}$ | 使湿润红石蕊变蓝 |
| 二氧化碳 $\ce{CO2}$ | 使石灰水变浑浊 |
| 氢气 $\ce{H2}$ | 点燃有“pop”声 |
| 氧气 $\ce{O2}$ | 使带火星木条复燃 |

## 有机分析检验

这些检验及其阳性结果是 syllabus 中要求的实验分析内容。

| 检验 | 阳性现象 | 推断 |
| --- | --- | --- |
| Fehling's reagent | 橙色/红色沉淀 | 醛基 |
| Tollens' reagent | 银镜或黑色沉淀 | 醛基 |
| alkaline aqueous iodine | 黄色三碘甲烷沉淀 | $\ce{CH3CO-}$ 或 $\ce{CH3CH(OH)-}$ 基团 |
| acidified $\ce{KMnO4}$ | 紫色变无色 | 可被氧化的基团 |

完整的羰基分析通常先用 2,4-DNPH 确认羰基，再用 Fehling's 或 Tollens' 判断醛或酮，最后用三碘甲烷检验 $\ce{CH3CO-}$ 基团。

## 处理不熟悉的离子

未知物可能含有 notes 中未列出的离子。这时要仔细记录现象，并作出一般性结论（例如“含碳酸根，因为加酸冒气泡且气体使石灰水变浑浊”），而不是乱猜某个具体离子。

## 常见错误

- 还没记下现象就写下结论。
- 忘记检验过量中的溶解性，导致 $\ce{Al^{3+}}$ 和 $\ce{Zn^{2+}}$ 无法区分。
- 漏写“no change”这类能排除可能性的阴性观察。
- 只点名气体而不写对应检验现象。
- 把有机检验顺序用错，例如没确认羰基就用 Fehling's reagent。
