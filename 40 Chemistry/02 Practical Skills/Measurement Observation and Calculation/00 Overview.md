# Measurement, Observation and Calculation

This note collects the practical habits behind Paper 3 quantitative and observational work: collect usable data, present it clearly, and calculate from it without inventing precision.

## Syllabus Route

Paper 3 focuses on manipulation, measurement and observation; presentation of data and observations; and analysis, conclusions and evaluation. Common quantitative contexts include titration, rates, gravimetric work, thermometric work, and gas volume experiments.

## Measurement Habits

- Record raw readings to the precision of the apparatus.
- Keep all readings of the same quantity to the same decimal place.
- For titrations, obtain concordant titres before calculating a mean.
- For rates work, define the observable end point before timing.
- For thermometric work, note the initial temperature, final or maximum temperature, and temperature change.
- For gas volume work, check for leaks and record volume units clearly.

## Tables and Graphs

- Put raw data and processed data in a single clear table where practical.
- Use headings with units, such as `volume / cm3` or `volume (cm3)`.
- Plot variables on labelled axes with suitable scales.
- Draw a line or curve of best fit rather than joining every point.
- Identify anomalous points instead of hiding them.
- Use a large triangle when finding the gradient of a straight-line graph.

## Calculations

- Show key working, especially mole ratios, concentration calculations, percentage errors, and graph-gradient work.
- Keep units attached to intermediate values.
- Use appropriate significant figures: usually the same as, or one more than, the least precise measured value used.
- Check that the answer has a plausible size and sign.

## Common Weaknesses

- Rounding intermediate values too early.
- Mixing raw readings and calculated values without clear headings.
- Reporting calculated concentrations to too many significant figures.
- Calling every scatter point an "error" instead of considering uncertainty, anomaly, or method limitation.
