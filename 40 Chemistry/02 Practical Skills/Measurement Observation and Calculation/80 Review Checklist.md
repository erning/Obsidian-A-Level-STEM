# Measurement, Observation and Calculation - Review Checklist

## Recall

- [ ] I can state the precision of a burette, pipette, thermometer, balance and measuring cylinder.
- [ ] I can state the rule for significant figures in a calculated result.
- [ ] I can define systematic and random error.
- [ ] I can define maximum uncertainty and percentage error.
- [ ] I can state the accepted table-heading conventions for units.

## Skills

- [ ] I can record every reading of a quantity to the same precision.
- [ ] I can design a results table with headings and units before taking readings.
- [ ] I can plot a graph with correct axes, scales, points and line of best fit.
- [ ] I can calculate a gradient using two well-separated points.
- [ ] I can calculate a percentage error for a measured change.
- [ ] I can identify anomalous data and handle it correctly.

## Common Errors

- [ ] I do not mix precisions within one column.
- [ ] I do not quote calculated values to more significant figures than the data justify.
- [ ] I do not join every point on a graph.
- [ ] I do not use a tiny triangle for a gradient.
- [ ] I do not write "human error" instead of naming a specific systematic or random error.
- [ ] I do not forget that a temperature change has uncertainty from both readings.

## Self-Checks

- [ ] Given a set of titres, I can find the anomalous value and the correct mean.
- [ ] Given raw data, I can state the correct number of significant figures for an answer.
- [ ] Given a graph, I can calculate a gradient with units.
- [ ] Given an instrument, I can state its maximum uncertainty and the percentage error of a reading.

## Connections

- [ ] I can link measurement precision to apparatus choice in *Experimental Design and Safety*.
- [ ] I can link graph and error work to *Planning, Analysis and Evaluation*.
- [ ] I can link observation recording to *Qualitative Analysis*.

## Ready When

- [ ] I can take a set of readings and present them faultlessly in a table.
- [ ] I can draw and interpret a graph to find a gradient or intercept.
- [ ] I can quantify uncertainty as a percentage error and judge whether it is acceptable.
