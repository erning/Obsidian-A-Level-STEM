# Experimental Design and Safety - Review Checklist

## Recall

- [ ] I can define independent, dependent and control variables.
- [ ] I can state what a prediction and a control experiment are.
- [ ] I can list the precision of common apparatus (burette, pipette, thermometer, balance, measuring cylinder).
- [ ] I can define hazard and risk and distinguish them.
- [ ] I can describe standard laboratory practice: standard solutions, weighing by difference, concordant titres, heating to constant mass.

## Skills

- [ ] I can identify the independent and dependent variables from a question.
- [ ] I can name control variables and state how each is kept constant.
- [ ] I can choose apparatus suited to a measurement's required precision.
- [ ] I can match a safety precaution to the actual chemical hazard.
- [ ] I can write a results table with headings and units before collecting data.
- [ ] I can describe a control experiment that tests causation.

## Common Errors

- [ ] I do not name a control variable without saying how it is held constant.
- [ ] I do not give a safety precaution that does not match the hazard.
- [ ] I do not change two variables at once.
- [ ] I do not say "repeat for accuracy" without stating what is averaged or checked.
- [ ] I do not list apparatus without explaining why its precision suits the task.

## Self-Checks

- [ ] Given a question, I can separate variables into independent, dependent and control.
- [ ] Given a method, I can spot the hazard and give the matching precaution.
- [ ] Given a weak plan, I can rewrite a control variable description precisely.
- [ ] Given a measurement need, I can pick the right apparatus and precision.

## Connections

- [ ] I can link design choices to reliable measurement in *Measurement, Observation and Calculation*.
- [ ] I can link a design into a full Paper 5 plan in *Planning, Analysis and Evaluation*.
- [ ] I can link safe handling of unknowns to *Qualitative Analysis*.

## Ready When

- [ ] I can write a complete plan: prediction, variables, method, table, safety, one reliability check.
- [ ] I can justify each apparatus choice by precision.
- [ ] I can identify the single largest design weakness in a described method.
